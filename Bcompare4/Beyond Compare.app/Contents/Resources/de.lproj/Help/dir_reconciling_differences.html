Unterschiede in der Ordnervergleichsansicht auflösen

Eine Ordnervergleichssitzung bietet Befehle wie Kopieren, Verschieben, Löschen und Synchronisieren, um Unterschiede auf Ordnerebene aufzulösen.  Sie ist zudem eine Plattform zum Starten von Dateiansichten zum Aufspüren und Auflösen von Unterschieden innerhalb von Dateien.

Aktionen benötigen im Allgemeinen eine Auswahl.  Sie können bestimmte Dateien oder Ordner auf einer oder beiden Seiten auswählen, und dann eine Aktion auf ihnen ausführen lassen.  (Wenn Sie den gesamten Inhalt der Basisordner synchronisieren möchten, ist stattdessen die Nutzung einer Ordnersynchronisation in Erwägung zu ziehen.)

Um Objekte in der Ordnervergleichsansicht auszuwählen

Klicken Sie auf ein Objekt auf einer der Seiten der Anzeige um es auszuwählen.  Sie können auf die mittlere Spalte klicken, um beide Seiten auf einmal auszuwählen.  Halten Sie die Strg-Taste gedrückt und klicken Sie Objekte an, um sie zur Auswahl hinzuzufügen.  Um einen Bereich von Objekten auszuwählen, drücken sie die Maustaste auf dem ersten Objekt, und ziehen Sie die Maus bis zum letzten auszuwählenden Objekt; dann lassen Sie die Maustaste wieder los.  Wenn Sie mit der Maus über die Mittelspalte ziehen, werden Einträge auf beiden Seiten ausgewählt.

Um alle Objekte einer Seite auszuwählen

Wählen Sie Bearbeiten > Alles auswählen (oder Bearbeiten > Alle Dateien auswählen ).  Halten Sie die Umschalt-Taste gedrückt, und drücken Sie die Taste Pfeil-links, um die Auswahl auf die linke Seite zu beschränken.  Drücken Sie stattdessen die Taste Pfeil-rechts, um die Auswahl auf die rechte Seite zu beschränken.

Um unterschiedlich benannte Dateien zu vergleichen

Klicken Sie die erste Datei an, halten Sie die Strg-Taste gedrückt und wählen Sie die zweite Datei aus; wählen Sie dann Aktionen > Öffnen.  Um stattdessen die Tastatur zu verwenden, gehen Sie zur ersten Datei, drücken die Leertaste, gehen zur zweiten Datei und drücken erneut die Leertaste.

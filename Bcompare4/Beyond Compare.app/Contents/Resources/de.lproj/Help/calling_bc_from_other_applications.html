Beyond Compare von anderen Anwendungen heraus aufrufen

Die Installation von Beyond Compare erstellt, wie allgemein üblich, Verknüpfungen zur Hauptprogrammdatei, wie:

 C:\Programme\Beyond Compare 3\BCompare.exe

Wenn Sie Beyond Compare von einer Batch-Datei oder einem Versionskontrollsystem aus aufrufen, empfehlen wir, "BComp" anstatt "BCompare.exe" zu verwenden.  BComp ist ein Hilfsprogramm, welches Beyond Compare startet und so lange wartet bis es geschlossen wird.  Weitere technische Details finden Sie in der Kommandozeilenreferenz.

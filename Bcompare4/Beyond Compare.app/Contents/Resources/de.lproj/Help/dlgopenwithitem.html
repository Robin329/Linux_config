"Öffnen mit"-Objekt bearbeiten

Geben Sie einen Namen im Feld Beschreibung ein, der im "Öffnen mit"-Menü erscheinen wird.

Um ein Tastaturkürzel zu definieren, gehen Sie auf das Eingabefeld Kürzel und drücken Sie die gewünschte Tastenkombination.  Klicken Sie die Schaltfläche Entfernen an, um ein zugeordnetes Kürzel zu entfernen.

Geben Sie zur Ausführung der Anwendung eine geeignete Befehlszeile ein.  Sie darf zur Anwendung passende Befehlszeilenschalter enthalten.  Die folgenden Variablen werden dabei mit den angegebenen Informationen ersetzt:

%f

Absoluter Pfad und Dateiname

%l

(kleines L) Aktuelle Zeilennummer in Dateiansicht

%n

nur Dateiname

%p

Absoluter Pfad

%x

Erweiterung, inklusive "."

%b

Dateiname ohne Erweiterung

%F

Pfad und Dateiname relativ zum Arbeitsordner

%P

Pfad relativ zum Arbeitsordner

"1" und "2" kann an das Ende der Variablen gehängt werden, um sich auf die linke/erste oder rechte/zweite Datei zu beziehen (z.B. "BCompare.exe %F2 %F1").  In jedem Fall, außer bei einem zweiten %f oder %l (für Rückwärtskompatibilität), wird sich das Auslassen einer Zahl auf die erste Datei beziehen, egal wieviele Male sie in der Befehlszeile aufgeführt ist.

Das Eingabefeld Arbeitsordner ermöglicht Ihnen anzugeben, wo das Programm laufen soll (der "Ausführen in"-Ordner).  Sollte das Feld leer sein, wird das Programm im selben Arbeitsordner laufen wie Beyond Compare.  Sie können einen bestimmten Ordner angeben oder einen der speziellen Werte <Elternordner> und <Basisordner> aus der aufklappbaren Liste auswählen.  <Elternordner> wird den Elternordner der ausgewählten Datei verwenden, während <Basisordner> den Basisordner der ausgewählten Datei verwendet (bzw. den Elternordner in einer Dateiansicht).

Verwenden Sie das Eingabefeld Pfadtrennzeichen, um alle Pfadtrennzeichen mit der angegebenen Zeichenfolge zu ersetzen.  Wenn das Feld leer bleibt, werden die üblichen Trennzeichen verwendet (z.B. umgekehrter Schrägstrich/Backslash (\) unter Windows).

Die Kontrollkästchen Akzeptiert Dateien und Akzeptiert Ordner steuern, ob die Anwendung Dateien, Ordner oder beides annimmt.  Tipp: Sie können Beyond Compare selbst als zusätzliche Anwendung für Ordner konfigurieren.  Auf diese Weise können Sie Ordner auswählen und in einer weiteren Instanz von Beyond Compare vergleichen.

Aktualisieren wenn fertig führt eine Schnellaktualisierung aus, sobald Beyond Compare mit der Abarbeitung der "Öffnen mit"-Liste fertig ist.

Wenn das Kontrollkästchen Mehrfachinstanzen markiert ist, können Sie mehrfache Dateien oder Dateipaare auswählen, und die Operation auf allen einmal pro Datei oder Dateipaar ausführen.  Mit dieser Option wird Beyond Compare prüfen, ob eine zweite Datei auf der Befehlszeile gegeben ist (z.B. "%x2").  Wenn ja, wird die Anwendung auf Paaren zu einer Zeit ausgeführt; ansonsten wird das Paar getrennt, und einmal für jede einzelne Datei des Paares ausgeführt.

Auf das Ende der vorherigen Instanz warten veranlasst, dass Beyond Compare mit der Ausführung des nächsten Befehls wartet bis der vorherige Befehl abgeschlossen ist.

Grammatiken

Die Spezifikation eines Dateiformates kann eine Grammatik-Definition enthalten.  Diese wird für Syntaxfärbung verwendet, und unterstützt dabei, welche Unterschiede als wichtig angesehen werden sollen.

Unser Standard-Dateiformat für beispielsweise C/C++ Quellcode enthält eine Grammatik, die Schlüsselworte, Kommentare und Literale erkennt.  Indem Sie Unterschiede in Kommentaren als unwichtig kennzeichnen, können Sie sich leichter auf wichtige Unterschiede konzentrieren.

Eine Grammatikdefinition in Beyond Compare ist recht schlicht.  Es werden keine verschachtelten oder kontextspezifischen Definitionen unterstützt.  Daher ist die Behandlung von "tagged" Sprachen wie HTML und PHP nicht absolut perfekt.  Auf der anderen Seite ist es ein großer Schritt weiter, verglichen mit der Funktionalität von Beyond Compare 2.

Um eine Grammatik zu bearbeiten, wählen Sie Extras > Dateiformate und suchen Sie das entsprechende Dateiformat.  Wechseln Sie zum Tab-Reiter Grammatik und bearbeiten das Grammatik-Element.

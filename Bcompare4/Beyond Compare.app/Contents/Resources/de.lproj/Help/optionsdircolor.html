Ordneransichtoptionen für Farben, Schriftarten

Extras > Optionen > Farben, Schriftarten > Ordneransichten

Diese Optionen ermöglichen die Einstellung von Farben und Schriftarten, die in den Ordneransichten verwendet werden.

Schriftart

Standardmäßig wird für Ordneransichten dieselbe Schriftart verwendet, die auch das Betriebssystem benutzt.  Um sie zu ändern, entfernen Sie die Markierung aus dem Kontrollkästchen Systemschrift verwenden, und drücken Sie die Schaltfläche Schriftart.

Auswahl

Das Element Auswahl steuert die Behandlung von ausgewähltem/markiertem Text.  Standardmäßig verwendet Beyond Compare eine mittelblaue Farbe für Auswahlen, so dass die Färbung von Unterschieden weiterhin gut zu erkennen ist.  Wenn Sie die Standardbehandlung des Betriebssystems für Markierungen bevorzugen, markieren Sie das Konstrollkästchen wie System.  Ausgewählte Dateien werden dann die Unterschiedfärbung verlieren, obwohl die Farbkennung links des Dateinamens weiterhin passend gefärbt sein wird.

Verschiedenes

Wählen Sie die Farben, die für Gefilterte Objekte verwendet werden (wenn Filter unterdrücken ausgewählt ist), und für Objekte, deren Vergleichsstatus Unbekannt ist.  Markieren Sie das Kontrollkästchen Streifen verwenden, um den Hintergrund zeilenweise abwechselnd einzufärben.

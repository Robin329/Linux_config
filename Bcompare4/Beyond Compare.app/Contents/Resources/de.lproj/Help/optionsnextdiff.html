Optionen für nächsten Unterschied

Extras > Optionen > Nächster Unterschied

Setzen Sie diese Optionen nach Ihren Wünschen, wie Sie Unterschiede suchen und prüfen möchten.

Beim Laden neuer Dateien zum ersten Unterschied springen

Wenn markiert, dann erfolgt beim Start eines neuen Dateivergleichs unmittelbar ein Sprung zum ersten Unterschied.

Nach dem Kopieren auf eine andere Seite zum nächsten Dateiunterschied springen

Wenn markiert, dann erfolgt ein Sprung zum nächsten unterschiedlichen Objekt, nachdem das aktuelle Objekt auf die andere Seite kopiert wurde.

"Nächste unterschiedliche Dateien" auf aktuellen Ordner beschränken

Wenn markiert, dann findet der Befehl Nächste unterschiedliche Dateien nur Dateien mit dem selben Elternordner.

Wieder zum ersten Unterschied springen

Wenn markiert, erfolgt ein Rücksprung zum ersten Unterschied innerhalb des Vergleichs, nachdem das Ende erreicht wurde.

Mitteilungsbereich anzeigen

Wenn nicht markiert, dann wird der Mitteilungsbereich (und das akustische Signal) unterdrückt, wenn kein Unterschied gefunden wurde.

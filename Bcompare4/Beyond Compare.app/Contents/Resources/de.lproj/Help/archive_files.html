Archivdateien

Bei Archiven, z.B. ZIP, handelt es sich um Dateien, die wiederum andere Dateien enthalten.  Beyond Compare stellt diese als Ordner mit einem Reißverschlusssymbol (=Zipper) dar, behandelt sie jedoch beim Kopieren und Vergleichen als Dateien.  Sobald Sie das Archiv per Doppelklick geöffnet bzw. aufgeklappt haben, werden die enthaltenen Dateien verglichen.  Viele der Formate unterstützen sowohl lesenden als auch schreibenden Zugriff, so dass Sie sämtliche üblichen Dateioperationen (Vergleichen des Inhaltes, Kopieren, Umbenennen usw.) innerhalb von Archiven anwenden können.

Gängige Archivformate

 7-zip (*.7z;*.7z.001)

 Beyond Compare Snapshot (*.bcss;*.bcs)

 BZip (*.bz;*.bz2)

 BZipped Tar (*.tbz;*.tbz2;*.tar.bz2)

 Compiled HTML Help (*.chm)

 Debian Package (*.deb)

 Disk Image (*.img;*.iso)

 GZip (*.gz)

 GZipped Tar (*.tgz;*.tar.gz)

 Microsoft Cabinet (*.cab)

 RAR (*.rar)

 Red Hat Package (*.rpm)

 Tar (*.tar)

 Windows Imaging Format (*.wim;*.swm)

 Xz (*.xz)

 Zip (*.zip;*.zipx;*.jar;*.ear;*.war;*.bcpkg)

 

Um eine vollständige Liste aller Archivformate zu erhalten oder die ihnen zugeordneten Dateierweiterungen zu ändern, wählen Sie Extras > Optionen und wechseln Sie auf die Seite Archivtypen.

Beyond Compare Schnappschüsse

Diese Dateien sind ähnlich wie andere Archive.  Allerdings enthalten sie lediglich die Dateinamen, Datumsangaben und Größen, ohne jedoch den Dateiinhalt selbst zu enthalten.  Da sie sehr klein sind, können Sie einen Schnappschuss einer gesamten Festplatte speichern und diesen zu einem späteren Zeitpunkt zum Vergleichen heranziehen.

Total Commander "Packer"-Plug-ins

Beyond Compare ist auch in der Lage, Total Commander "Packer"-Plug-ins zur Unterstützung zusätzlicher Archivformate einzubinden.  Beispielsweise stehen Plug-ins zur Verfügung, die ISO-Disk-Images (.iso) und Microsoft Installationspakete (.msi) unterstützen.

Um eines dieser Drittanbieter-Plug-ins hinzuzufügen, klicken Sie auf der Seite Archivtypen auf die Schaltfläche Plug-in hinzufügen und folgen Sie den dortigen Anweisungen.

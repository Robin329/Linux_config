FTP-Profil-Übertragungseinstellungen

FTP-Profil > Übertragen

Diese Einstellungen steuern wie Dateien von und zu dem FTP-Server übertragen werden.

Transfertyp

FTP-Standorte können Daten auf zwei Arten übertragen:  ASCII und Binär.  ASCII wird für die Übertragung von Textdateien verwendet, die keine Sonderzeichen enthalten.  Der FTP-Server wird automatisch Anpassungen hinsichtlich Zeilenendungen vornehmen, die im ASCII-Modus notwendig sind.  Der ASCII-Modus bewältigt auch Übertragungen von Großrechnern, indem EBCDIC in ANSI konvertiert und Zeilenendezeichen am Ende von Übertragungssätzen angefügt werden.  Binär wird für Übertragungen aller anderen Dateien verwendet, inkl. Bilder, Musik, Word Dokumente usw.  Im Binär-Modus werden während der Übertragung keine Änderungen an den Dateien vorgenommen.

Das Setzen einer der Transfertypen ASCII oder Binär erzwingt die Übertragung aller Dateien in dem gewählten Modus.  Wenn der Transfertyp auf Autom. gestellt ist, wird Beyond Compare automatisch den Transfertyp anhand der Dateierweiterung bestimmen.

Zeitstempel beim Hochladen kopieren

Markieren Sie diese Option, um den entfernten Zeitstempel an den Zeitstempel der Quelle anzupassen.  Entfernen Sie diese Markierung nur, wenn es Probleme mit dem Server gibt.

Unix-Rechte beim Hochladen kopieren

Markieren Sie diese Option, um die entfernten Unix-Rechte an die Unix-Rechte der Quelle anzupassen.  Entfernen Sie diese Markierung nur, wenn es Probleme mit dem Server gibt.

Übertragungen komprimieren (MODE Z)

Markieren Sie diese Option, um Komprimierung zur Beschleunigung der Übertragung anzuwenden, sofern dieses vom Server unterstützt wird.  Dies könnte allerdings Übertragungen über schnelle Verbindungen verlangsamen.

Download beschränken (Kbps), Upload beschränken (Kbps)

Spezifizieren Sie, wieviel Bandbreite Beyond Compare für jede Verbindung beanspruchen darf.

Schnellere Uploads zu älteren OpenSSH-Servern erzwingen

Markieren Sie diese Option, um größere Übertragungspuffer und Pipelining auf OpenSSH-Servern (älter als OpenSSH 3.9) zu verwenden.  Dies kann die Übertragungsgeschwindigkeit signifikant erhöhen, ist jedoch nicht mit allen Servern kompatibel, und Fehler könnten nicht unbedingt offensichtlich werden.  Machen Sie einen binären Inhaltsvergleich, um die ersten Übertragungen zu überprüfen, nachdem Sie diese Option aktiviert haben.

Hinweis:  Sollten Sie Unterstützung für die Einstellungen auf dieser Seite benötigen, dann wenden Sie sich bitte an Ihren Systemadministrator.

Verwaltung der Profile

Extras > Profile

Wählen Sie Extras > Profile, um alle Profile aufzulisten und sie zu verwalten.

Zusätzlich zu den benannten Profilen gibt es das <Standard> FTP-Profil, dessen Einstellungen für selbst benannte FTP-Profile genutzt werden können.

Wenn Sie bestimmte Profile erstellen, wie Dropbox und OneDrive, ist es notwendig, dass Sie Beyond Compare Zugriff gewähren.  Typischerweise beinhält dies das Einloggen in den Service und das Klicken einer Schaltfläche.  Dropbox verlangt außerdem, dass Sie einen Code aus dem Webbrowser kopieren und in Beyond Compare einfügen.

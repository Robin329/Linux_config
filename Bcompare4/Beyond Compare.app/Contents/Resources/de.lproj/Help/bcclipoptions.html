Zwischenablagevergleich Optionen

Diese Optionen steuern das Verhalten des Zwischenablagevergleichs.

Bei Windows-Start ausführen

Führt automatisch den Zwischenablagevergleich aus, wenn Windows startet.

Tastaturkürzel aktivieren

Aktiviert das Strg+Alt+C Tastaturkürzel zum Vergleichen der beiden letzten Zwischenablageeinträge.

Zugeordnete Dateiformatmaske

Setzt eine einfache Dateimaske, wie zum Beispiel *.java, die zur Auswahl eines geeigneten Dateiformats für Textvergleiche verwendet wird.  Wenn es leer gelassen wird, so wird das Standard-Dateiformat verwendet.

Minimale Abklemmlänge

Setzt die minimale Zeichenanzahl, aus denen ein Text bestehen muss, der gemerkt werden soll.  Kann angewendet werden, um das Speichern von Passwörtern oder anderen kurzen Texten zu vermeiden.  Setzen Sie 0 um alles zu merken.

Speicherbeschränkung (Clips)

Setzt die maximale Anzahl von Einträgen, die gehalten werden sollen.

Speicherbeschränkung (MB)

Setzt die maximale Speichergröße, die für alle Einträge verwendet werden soll.

Hinweis:  Der Zwischenablagevergleich wird versuchen, die letzten zwei Zwischenablageeinträge zu behalten, sogar wenn das Speichernutzungslimit überschritten wird.

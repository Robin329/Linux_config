Ausrichtungseinstellungen für Text-Merge

Sitzung > Sitzungseinstellungen > Ausrichtung

Die Einstellungen des Tab-Reiters Ausrichtung steuern den Algorithmus zur Zeilenausrichtung im Vergleichsprozess.

Markieren Sie das Kontrollkästchen Unterschiede niemals aneinander ausrichten, um Zeilen mit wichtigen Unterschieden als hinzugefügte oder gelöschte Blöcke darzustellen, statt als geänderten Text.

Wählen Sie Nicht ausgerichtet, um Dateien zu vergleichen, ohne die Zeilen basierend auf ihrem Inhalt auszurichten.

Wählen Sie Standard-Ausrichtung, um die Dateien aneinander auszurichten, indem sukzessive kleinere Abschnitte jeder Datei miteinander verglichen werden.  Teile der Ausrichtung können bereits angezeigt werden, bevor der gesamte Vergleich beendet ist.

Wählen Sie Myers 0(ND) Ausrichtung, um Dateien unter Verwendung eines gewöhnlichen LCS-Algorithmus (Longest Common Subsequence = Längste gemeinsame Teilfolge) auszurichten.  Dieses kann in bestimmten Fällen zu besseren Übereinstimmungen führen, z.B. bei umfangreichen Einfügungen, oder wenn die Dateien viele sich wiederholende Texte enthalten.  Da die Dateien auf einmal verglichen werden, benötigen größere Dateien mehr Zeit als die Standard-Methode.  Zudem werden die Dateien nicht angezeigt, bevor der Vergleich nicht beendet ist.  Diese Methode unterstützt keinen Ähnlichkeitsvergleich, so dass Unterschiede in Blöcken gruppiert werden.

Wählen Sie Patience Diff Ausrichtung, um Dateien unter Verwendung des Bram-Cohen-Algorithmus auszurichten.

Der Übereinstimmungssuchbereich spezifiziert die maximale Zeilenanzahl, die der Algorithmus überprüfen wird, wenn er nach einer Übereinstimmung für eine bestimmte Zeile sucht.  Eine Erhöhung kann die Ausrichtung verbessern, besonders wenn es größe Lücken gibt.  Natürlich könnte der Vergleich auch mehr Zeit benötigen.

Markieren Sie das Kontrollkästchen Näheangleichung verwenden, um die übrigen ungleichen Zeilen, basierend auf ihrer Ähnlichkeit, aneinander auszurichten.

Hinweis:  Unabhängig von diesen Einstellungen, produzieren die Ausrichtungsalgorithmen manchmal unerwünschte Resultate.  Um zwei Zeilen manuell auszurichten, führen Sie einen Rechtsklick auf einer Zeile aus, wählen Sie Ausrichten mit und klicken Sie dann die zweite Zeile an.

Ein Merge-Konflikt ist

Diese Einstellungen definieren den Umfang von Konflikten.  Ein Konflikt tritt auf, wenn im selben Textabschnitt Veränderungen sowohl auf der linken als auch auf der rechten Seite auftreten.

Wählen Sie nur unterschiedliche geänderte Zeilen, um Konflikte auf das zu beschränken, wo jede Zeile auf beiden Seiten geändert wurde.  Wählen Sie links- und rechtsseitige Änderung, getrennt durch folgende Zeilenanzahl, um eine maximale Zeilenanzahl zwischen Änderungen, die als Konflikt angesehen werden, zu spezifizieren.  Auf diese Weise können nahe beieinander liegende, links- und rechtsseitige Änderungen für weitere Überprüfung mit direkten Konflikten markiert werden.

Spaltendefinition bearbeiten

Der Dialog für Spaltenbehandlung erlaubt Ihnen Optionen für jede Vergleichsspalte zu setzen.  Es gibt einen Teildialog dieses Dialoges, der für die Einstellung der Standardbehandlung für Spalten verwendet wird.

Geben Sie Name an, der im Vergleichspaltenkopf angezeigt werden soll.  Wenn kein Name angegeben wurde, wird ein automatisch generierter verwendet.

Markieren Sie das Kontrollkästchen Schlüssel, um Daten dieser Spalte zur Steuerung der Sortierung und Vergleichsausrichtung zu verwenden.  Sollten Sie mehr als eine Schlüsselspalte haben, wird den obersten Spalten in der Liste die Sortierpriorität gewährt.

Markieren Sie das Kontrollkästchen Standard verwenden, um die Standardspaltenbehandlung zu verwenden, wie sie für diese Sitzung definiert ist.

Wählen Sie den passenden Typ für die Spalte:

Erkannt

Erkennt automatisch den Spaltentyp, basierend auf den Daten in der Datei.

Boolean

Spaltenwerte sind entweder "True" (wahr) oder "False" (falsch), oder numerisch.  Eine Null ist gleichbedeutend mit "False" und jeder Wert ungleich 0 ist äquivalent zu "True".

Datum

Spalte enthält Datums- bzw. Zeitangaben, in jedem gängigen Format.  Der Vergleich ignoriert Formatierungsunterschiede.

Numerisch

Spaltenwerte werden als "reelle Zahlen" interpretiert.  Der Vergleich ignoriert Formatierungsunterschiede.  Zum Beispiel: 0,00 und 0 werden als identisch angesehen.

Text

Spalte enthält Text bzw. Zeichenfolgen.

Markieren Sie das Kontrollkästchen Unwichtig, um zu kennzeichnen, dass diese Spalte für den Vergleich unwichtig ist.  (Unwichtige Unterschiede werden blau gefärbt statt rot, oder sie werden bei Verwendung der Option Unwichtige Unterschiede ignorieren vollständig ignoriert.)

Verwenden Sie Datumstoleranz, damit Datums- bzw. Zeitangaben dieser Spalte um den angegebenen Betrag abweichen dürfen, bevor sie als wichtiger bzw. signifikanter Unterschied angesehen werden.

Setzen Sie einen Wert für Numerische Toleranz, damit die Zahlenwerte dieser Spalte um den angegebenen Betrag abweichen dürfen, bevor sie als wichtiger bzw. signifikanter Unterschied angesehen werden.

Die Option Text wichtig, außer betrifft Spalten, die als Text erkannt oder definiert wurden.  Markieren Sie das Kontrollkästchen Groß-/Kleinschr. (ABC = abc), wenn Unterschiede hinsichtlich Groß-/Kleinschreibung als unwichtig angesehen werden sollen.  Markieren Sie das Kontrollkästchen Leerzeichen (Whitespace), wenn Unterschiede hinsichtlich Leerzeichen (Anzahl der Leerzeichen vor, hinter oder zwischen Wörtern) als unwichtig angesehen werden sollen.

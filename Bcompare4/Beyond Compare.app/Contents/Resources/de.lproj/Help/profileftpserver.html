FTP-Profil-Server-Einstellungen

FTP-Profil > Server

Groß-/Kleinschreibung in Dateinamen ist signifikant

Markieren Sie diese Option, um Übereinstimmung in Groß-/Kleinschreibung für die Ausrichtung im Vergleichsprozess erforderlich zu machen.

Kodierung

Bestimmen Sie die Zeichenkodierung, die der Server für Verzeichnisauflistungen und Befehle verwendet.

Zeitzone

Bestimmen Sie die Zeitzone des Servers.  Zeitstempel werden automatisch in lokale Zeit umgewandelt.

Benutzerdef. Login-Befehle

Bestimmen Sie hier zusätzliche Befehle, die beim Verbinden an den Server gesendet werden sollen.  Nicht unterstützt von SFTP (SSH) Verbindungen.

Hinweis:  Sollten Sie Unterstützung für die Einstellungen auf dieser Seite benötigen, dann wenden Sie sich bitte an Ihren Systemadministrator.

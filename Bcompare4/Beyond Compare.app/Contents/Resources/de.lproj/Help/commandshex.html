Befehle für Hex-Vergleich

Im Folgenden ist eine Beschreibung der Befehle für Hex-Vergleich als Ergänzung zu den Allgemeinen Befehlen aufgeführt.  Der Großteil dieser Befehle ist als Schaltflächen in der Symbolleiste verfügbar.

Um Ihre eigenen Tastaturkürzel zu definieren und Befehle in Menüs und Symbolleisten ein- und auszublenden, wählen Sie Extras > Optionen und wechseln Sie auf die Seite Symbolleisten usw..

Das Menü "Sitzung"

Sitzung speichern

Speichert die aktuelle Sitzung.

Sitzung speichern unter

Speichert die aktuelle Sitzung unter einem neuen Namen.

Sitzungseinstellungen

Öffnet den Dialog für Sitzungseinstellungen für den Hex-Vergleich.

Gesperrt

Sperrt oder entsperrt aktuelle Sitzung.

Sitzung zurücksetzen

Ersetzt die aktuelle Ansicht mit einem leeren Hex-Vergleich.

Seiten tauschen

Tauscht die linke und rechte Datei.

Dateien neu laden

Lädt die angezeigten Dateien erneut.  Wenn notwendig, erfolgt eine Abfrage, ob gespeichert werden soll.

Dateien erneut vergleichen

Vergleicht die angezeigten Dateien erneut.  Behält die Änderungen bei, jedoch nicht die Fähigkeit zum Rückgängigmachen.

Hex-Vergleichsbericht

Erstellt einen Hex-Vergleichsbericht.

Hex-Vergleichsinfo

Zeigt Vergleichsstatistiken an.

Dateien vergleichen mittels

Öffnet die angezeigten Dateien in einer neuen Sitzung unter Verwendung einer anderen Dateiansicht.

Elternordner vergleichen

Öffnet eine neue Ordnervergleichssitzung basierend auf den Elternordnern der aktuell geladenen Dateien.

Das Menü "Datei"

Datei öffnen

Öffnet eine existierende Datei im gewählten Bereich.

Datei vom Dateisystem öffnen *

Öffnet eine existierende lokale Datei im gewählten Bereich.

Datei anhand Profil öffnen *

Öffnet eine existierende Datei im gewählten Bereich anhand eines Profils.

Zwischenablage öffnen

Lädt den Inhalt der Zwischenablage in den gewählten Bereich.

Öffnen mit

Öffnet die Datei im gewählten Bereich mit einer der externen "Öffnen mit"-Anwendungen.

Öffnen mit >

  Zugeordnete Anwendung

Öffnet die Datei im gewählten Bereich mit der ihr zugeordneten Anwendung.

Öffnen mit >

  Textbearbeitung

Öffnet die Datei im gewählten Bereich mit dem integrierten Texteditor.

Datei speichern

Speichert die geänderte Datei aus dem gewählten Anzeigebereich.

Datei speichern unter

Speichert Datei aus dem gewählten Bereich unter neuem Namen.

Datei im Dateisystem speichern unter *

Speichert die Datei aus dem gewählten Bereich unter neuem Namen im Dateisystem.

Datei mittels Profil speichern unter *

Speichert Datei aus dem gewählten Bereich unter neuem Namen mittels eines Profils.

Explorer

Zeigt das Kontextmenü des Dateimanagers für die Datei im ausgewählten Bereich an.

Das Menü "Bearbeiten"

Rückgängig

Macht vorherige Aktion rückgängig.

Wiederholen

Wiederholt die zuvor rückgängig gemachte Aktion.

Nach rechts kopieren / Nach links kopieren *

Kopiert die ausgewählten Bytes auf die angegebene Seite.

Zur anderen Seite kopieren

Kopiert die ausgewählten Bytes auf die gegenüberliegende Seite.  Die Beschriftung und das Symbol erscheinen, abhängig von der aktuellen Seite, als Nach rechts kopieren oder Nach links kopieren.

Ausschneiden / Kopieren / Einfügen / Löschen

Standardbefehle für die Bearbeitung.

Alles auswählen

Wählt alle sichtbaren Bytes im aktuellen Bereich aus.

Das Menü "Suchen"

Nächste unterschiedliche Bytes

Positioniert auf nächste verschiedene Bytes.

Vorherige unterschiedliche Bytes

Positioniert auf vorherige verschiedene Bytes.

Nächste unterschiedliche Dateien

Öffnet das nächste Paar unterschiedlicher Dateien der Elternsitzung.  (nur in Kindsitzungen.)

Vorherige unterschiedliche Dateien

Öffnet das vorherige Paar unterschiedlicher Dateien der Elternsitzung.  (nur in Kindsitzungen.)

Datei nach rechts kopieren und nächsten Unterschied öffnen

Kopiert die gesamte linksseitige Datei nach rechts und öffnet die nächste unterschiedliche Datei der Ordnersitzung.  (nur in Kindsitzungen.)

Datei nach links kopieren und nächsten Unterschied öffnen

Kopiert die gesamte rechtsseitige Datei nach links und öffnet die nächste unterschiedliche Datei der Ordnersitzung.  (nur in Kindsitzungen.)

Suchen

Sucht nach übereinstimmendem Text.

Ersetzen

Ersetzt übereinstimmenden Text.

Weitersuchen

Findet das nächste Auftreten der gesuchten Zeichenfolge.

Vorheriges suchen

Findet das vorherige Auftreten der gesuchten Zeichenfolge.

Gehe zu

Positioniert sich auf ein bestimmtes Byte.

Das Menü "Ansicht"

Alle anzeigen

Zeigt alle Bytes an, mit oder ohne Unterschieden.

Unterschiede anzeigen

Zeigt lediglich ungleiche Bytes an.

Identische anzeigen

Zeigt lediglich Bytes ohne Unterscheide an.

Aktuelle Byte-Adresse

Zeigt aktuelle Byte-Adresse an.

Little-Endian-Werte

Listet jene Werte auf, bei denen das Byte mit den am wenigsten signifikantesten Stellen (niederwertigen Bits) zuerst gespeichert wird.

Big-Endian-Werte

Listet jene Werte auf, bei denen das Byte mit den signifikantesten Stellen (höchstwertigen Bits) zuerst gespeichert wird.

Byte-Adressen

Blendet Byte-Adressen ein oder aus.

Schriftart *

Wählt die Schriftart zur Anzeige der Daten aus.

Schriftartgröße erhöhen *

Erhöht die Hex-Schriftartgröße.

Schriftartgröße verringern *

Verringert die Hex-Schriftartgröße.

Schriftartgröße zurücksetzen *

Setzt Hex-Schriftartgröße zurück.

Layout

Konfiguriert Layoutoptionen.

Nebeneinander

Stellt die Bereiche links und rechts nebeneinander dar.

Untereinander

Stellt die Bereiche untereinander dar.

Miniaturansicht

Blendet die Miniaturansicht an der linken Seite der Anzeige ein oder aus.

Dateiinfo

Blendet Dateiinformation oberhalb der Bereiche ein oder aus.

Symbolleiste

Blendet die Symbolleiste ein oder aus.

* Standardmäßig ausgeblendet.

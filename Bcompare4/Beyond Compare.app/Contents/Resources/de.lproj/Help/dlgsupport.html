Hilfe > Unterstützung

Wenn Sie ein Problem mit Beyond Compare haben, wenden Sie sich gerne an uns!

Von jedem Fenster höchster Ebene wählen Sie Hilfe > Unterstützung aus, und folgen den Anweisungen, um eine Zusammenstellung Ihrer Programm- und Systemeinstellungen zu erstellen.  Hängen Sie die Zusammenstellung an eine E-Mail, inklusive einer Problembeschreibung, und senden Sie sie an uns: support@scootersoftware.com.

Möglicherweise möchten Sie einen Screenshot (Bildschirmabdruck) beifügen, der das Problem illustriert.  In diesem Falle folgen Sie diesen Schritten, um eine möglichst kleine E-Mail zu erstellen:

1.Drücken Sie Alt+Druck (oder auch Alt+PrintScreen) um eine Kopie des aktuellen Fensters in der Zwischenablage abzulegen.

2.Öffnen Sie das Microsoft Paint Programm (Start > Alle Programme > Zubehör > Paint) und drücken Sie Strg+V, um das Bild dort einzufügen.

3.Markieren Sie jenen Teil des Bildes, den Sie versenden möchten.

4.Wählen Sie Bearbeiten > Kopieren nach, ändern Sie den Dateityp in PNG (*.png) oder GIF (*.gif), und geben Sie einen Dateinamen ein.  (Wenn PNG und GIF nicht zur Verfügung stehen, nehmen Sie 256-Farben-Bitmap und komprimieren das Bild anschließend mit einem ZIP-Programm.)

5.Hängen Sie die Bilddatei an die E-Mail an, die Sie uns senden möchten.

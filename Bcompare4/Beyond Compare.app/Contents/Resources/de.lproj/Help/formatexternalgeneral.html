Allgemeine Einstellungen für externes Format

Externes Format > Allgemeines

Um Dateien unter Verwendung einer anderen Anwendung zu vergleichen, definieren Sie ein externes Format.

Die Maske sollte die Menge jener Dateitypen sein, die dadurch behandelt werden.

Die Befehlszeile für Schnellvergleich wird immer dann verwendet, wenn regelbasierte Vergleichsergebnisse benötigt werden.  Der Rückgabewert sollte 0 für passend und 1 für nicht passend sein.

Die Befehlszeile für Vergleichsansicht wird verwendet, wann immer eine Vergleichsansicht benötigt wird.

Die Befehlszeile für Schnell-Merge wird immer dann verwendet, wenn regelbasierte Merge-Ergebnisse benötigt werden.  Der Rückgabewert sollte 0 für passend und 1 für nicht passend sein.

Die Befehlszeile für Merge-Ansicht wird verwendet, wann immer eine Merge-Ansicht benötigt wird.

Die Beschreibung kann beliebigen Text enthalten.

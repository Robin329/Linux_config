Neuer Ordner

Aktionen > Neuer Ordner

Im Menü Aktionen ermöglicht Ihnen der Befehl Neuer Ordner die Erstellung eines neuen Ordners an der aktuellen Stelle.  Wenn Sie gerade eine Datei oder einen geschlossenen Ordner ausgewählt haben, wird der neue Ordner mit demselben Elternordner erstellt wie das ausgewählte Objekt.  Wenn Sie einen geöffneten Ordner ausgewählt haben, wird der neue Ordner als Unterordner des geöffneten Ordners erstellt.

Wenn beide Seiten ausgewählt sind, wird der neue Ordner auf beiden Seiten erstellt.

Sie können eine Archivdatei erstellen, unter Einbeziehung einer geeigneten Erweiterung (wie z.B. .ZIP) in dem Namen.

Um einen neuen Ordner zu erstellen und als Basisordner zu verwenden, geben Sie den Pfad des neuen Ordners in das linke oder rechte Eingabefeld ein, und drücken Sie die Enter-Taste.  Im nun erscheinenden Warnungsdialog klicken Sie die Schaltfläche Erstellen an.

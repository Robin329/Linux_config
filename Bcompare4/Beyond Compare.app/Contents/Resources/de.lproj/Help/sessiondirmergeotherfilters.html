Einstellungen für andere Filter für Ordnerzusammenführung

Sitzung > Sitzungseinstellungen > Andere Filter

Sie können Dateien von einer Ordnersitzung anhand anderer Kriterien als dem Namen ausschließen.  Klicken Sie die Schaltfläche Neu, um ein Andere Filter Objekt hinzuzufügen.

Hinweis:  Sobald irgendwelche Elemente anderer Filter definiert wurden, wird unterhalb der Ordneransicht eine Kommentarleiste erscheinen.  Sie können die Schaltfläche Andere Filter klicken, um diese Filter zu bearbeiten.

Hinweis:  Wenn ein Dateifilter erkennt, dass eine Datei auf einer Seite des Vergleichs versteckt werden soll, jedoch nicht die entsprechende Datei auf der anderen Seite, so werden beide sichtbar, und die eigentlich verborgene Datei wird türkis (blau-grün) gekennzeichnet.

Das Kontrollkästchen Geschützte Betriebssystemdateien ausschließen ist standardmäßig markiert.

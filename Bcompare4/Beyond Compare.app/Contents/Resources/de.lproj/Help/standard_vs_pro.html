Standard- contra Pro-Edition

Die Standard-Edition von Beyond Compare ist ein mächtiges, voll funktionsfähiges Vergleichsprogramm, welches nicht einfach nur zu einer "Light"-Version "abgespeckt" wurde.  Die Standard-Edition bietet den Vollbearbeitungsmodus für Dateien, volle Unicode-Unterstützung, Syntaxfärbung und vieles mehr.

Jedoch bietet die Pro-Edition zusätzlich diese weitergehenden Funktionen:

3-Wege-Merge

Vergleicht unabhängig gemachte Änderungen gegen einen gemeinsamen Vorläufer, um für Ordner oder einzelne Dateien einen neuen, zusammengeführten Inhalt zu erstellen.

Textersetzungen

Spezifiziert Text als unwichtig, wenn er auf einen bestimmten Wert auf der anderen Seite geändert wird.  Zum Beispiel: Sie können Apfel auf der linken Seite, verglichen mit Birne auf der rechten Seite, als unwichtigen Unterschied ansehen lassen.

Überschreiben der Dateiausrichtung im Ordnervergleich

Überschreibt wie Dateien in einer Ordnervergleichssitzung aneinander ausgerichtet werden.  Zum Beispiel: Sie könnten Ihre GIF-Dateien auf der einen Seite mit PNG-Dateien desselben Namens auf der anderen Seite vergleichen.

Integration von Versionskontrolle

Fügt Befehle zum Ein- und Auschecken von Dateien in/aus einem SCC-kompatiblen Versionskontrollsystem hinzu.

SFTP-Unterstützung

Statt von Drittanbieterprodukten abhängig zu sein, besitzt Beyond Compare Pro eine integrierte Unterstützung für sichere FTP-Protokolle (SFTP und FTP über SSL).

Registry-Vergleich

Vergleicht sowohl lokale als auch entfernte aktuelle Registrierdatenbanken, sowie Registry-Dateien.

 

Hinweis:  Die Evaluationsversion von Beyond Compare enthält prinzipiell beide Editionen, sowohl Standard als auch Pro.  Sie können den Pro-Modus im Dialog Hilfe > Über ausschalten, um Ihre Evaluierung auf Standardfunktionalität zu beschränken.

Ihr Lizenzschlüssel wird die entsprechende Edition für permanenten Gebrauch freischalten.  Wenn Sie sich für eine Lizenz für die Standard-Edition entscheiden, können Sie zu einem späteren Zeitpunkt ein Upgrade auf die Pro-Edition für die Preisdifferenz erwerben.

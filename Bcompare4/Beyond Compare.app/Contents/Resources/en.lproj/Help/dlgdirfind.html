Finding Filenames

Search > Find Filename

To locate a specific file in a folder view, pick Search > Find Filename .

Text to find

Enter the text to search for.

Match character case

Match only text with the same uppercase and lowercase characters.

Whole words only

Match on the whole words but not parts of words.

Regular expression

Indicates the search string is a PCRE regular expression.  Click on the Insert Regular Expression Element button to pick from a list of common regular expression terms.  Click the RE Examples button for more info.

Wrap search

When one end of the file is reached, continue searching at the other end.

Search up

Searches backwards, towards the top of the comparison.

Sides

Specify the side(s) of the comparison to search.

Use the Search menu's Find Next Filename and Find Previous Filename commands for finding additional occurrences of the search string.

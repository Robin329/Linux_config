Profile Management

Tools > Profiles

Pick Tools > Profiles to see and manage the entire list of profiles.

In addition to the named profiles, there is the <default> FTP profile whose settings can be used by named FTP profiles.

If you create certain profiles, such as Dropbox and OneDrive, you will be required to allow Beyond Compare to have access.  This typically involves logging into the service and clicking a button.  Dropbox additionally requires you to copy a code from the web browser and paste it into Beyond Compare.

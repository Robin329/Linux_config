Modifier un élément texte de remplacement

Spécifiez le Texte à rechercher et le texte Remplacer avec.

Cochez Respecter la casse de caractère pour faire coïncider le texte uniquement avec les mêmes caractères, majuscules ou minuscules.

Cochez Mots entiers uniquement pour empêcher la correspondance avec des parties de mots.

Cochez Expression régulière pour les interpréter comme des expressions régulières comme style de remplacement.  Cliquez sur le bouton Insérer un élément d'expression régulière pour choisir dans la liste des termes usuels d'expression régulière.  Cliquez sur le bouton Exemples d'ER pour plus d'informations.

Spécifiez le Côté du panneau de l'éditeur qui doit être cherché.

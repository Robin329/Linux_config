Paramètres de transfert de profil FTP

Profil FTP > Transférer

Ces paramètres contrôlent comment des fichiers sont transférés vers et à partir du serveur FTP.

Type de transfert

Les sites FTP peuvent transférer les données de deux façons :  ASCII ou Binaire.  L'ASCII est utilisé pour transférer des fichiers texte qui n'ont aucun caractère spécial.  Le serveur FTP fera automatiquement les ajustements de fins de ligne nécessaires au mode ASCII.  Le mode ASCII traite aussi les transferts des unités centrales en convertissant l'EBCDIC en ASCII.  Le mode Binaire est utilisé pour transférer tous les autres fichiers, incluant les images, la musique, les documents Word, etc.  Dans le mode Binaire, aucune modification n'est effectuée au fichier pendant le transfert.

Positionner le type de transfert à ASCII ou à Binaire forcera le transfert de tous les fichiers dans ce mode.  Si le type de transfert est Auto, Beyond Compare commutera automatiquement le mode en se basant sur l'extension de chaque fichier.

Copier l'horodatage lors de l'envoi

Cochez pour forcer l'horodatage distant à correspondre à l'horodatage de la source.  Ne le décochez que si cela cause un problème avec le serveur.

Copier les autorisations Unix lors de l'envoi

Cochez pour forcer les permissions de l'Unix distant à correspondre aux permissions de l'Unix source.  Ne le décochez que si cela cause un problème avec le serveur.

Compresser les transferts (MODE Z)

Cochez pour utiliser une compression afin d'accélérer les transferts lorsque cela est supporté par le serveur.  Cela peut réellement ralentir le transfert sur des connexions rapides.

Limiter le téléchargement (Kbps), Limiter l'envoi (Kbps)

Spécifiez combien de bande passante Beyond Compare peut consommer pour chaque connexion.

Forcer des envois plus rapides vers des serveurs OpenSSH anciens

Cochez pour utiliser des tampons de transfert plus grands pour les serveurs OpenSSH antérieurs à OpenSSH v3.9.  Cela peut substantiellement augmenter la vitesse de transfert, mais n'est pas compatible avec tous les serveurs, et les erreurs ne sont pas évidentes.  Utiliser une comparaison binaire de contenu pour vérifier les premiers transferts après l'avoir coché.

Note  Si vous avez besoin d'aide pour les paramètres de cette page, consultez votre administrateur système.

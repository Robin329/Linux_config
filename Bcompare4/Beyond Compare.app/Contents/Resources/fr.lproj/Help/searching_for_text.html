Rechercher du texte

Pour rechercher du texte dans une session de Comparaison de Textes, sélectionnez Rechercher > Recherche .  Entrez le texte ou un motif d'expression régulière dans le champ, ou sélectionnez une recherche précédente dans la liste déroulante.  Cliquez sur le bouton Insérer un élément d'expression régulière pour choisir dans la liste des termes usuels d'expression régulière.  Prenez-en une dans la liste pour l'insérer dans le champs de saisie.

Cliquez sur le bouton Suivant pour déplacer le curseur à la prochaine correspondance.  Si l'éditeur est en mode ligne, vous pouvez cliquer sur le bouton Sélectionner tout pour sélectionner toutes les lignes qui contiennent une correspondance.

Les options du panneau Recherche sont :

Option

Signification

Expression régulière

Interpréter le texte de recherche comme une expression régulière compatible Perl.

Respecter la casse de caractère

Trouver uniquement du texte avec la même casse de caractère (majuscule ou minuscule).

Mots entiers uniquement

Rechercher port ne trouvera pas ports, report ou importer.

Rechercher avec bouclage

Lorsqu'une fin du fichier est atteinte, continuer la recherche depuis l'autre bout.

Côtés

Sélectionner de quel(s) côté(s) de la comparaison rechercher.

Fermer automatiquement

Le panneau sera caché au démarrage de la recherche.

 

Sélectionnez Rechercher le suivant ou Rechercher le précédent pour continuer la recherche du même texte.

Pour rechercher et remplacer du texte, sélectionnez Remplacer .

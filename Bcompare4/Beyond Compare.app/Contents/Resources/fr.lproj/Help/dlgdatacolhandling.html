Modifier une définition de colonne

Le dialogue Traitement de colonne vous permet de définir les options pour chaque colonne de comparaison.  Un sous-ensemble de ce dialogue est utilisé pour définir le traitement par défaut de toutes les colonnes.

Spécifiez le Nom à afficher dans l'entête de la colonne de comparaison.  Si aucun n'est spécifié, un nom automatique est utilisé.

Cochez Clé pour que les données dans cette colonne contrôlent le tri et l'alignement de ligne.  Si vous avez plus d'une colonne clé, la priorité pour le tri sera donnée à la colonne la plus en haut dans la liste.

Cochez Utiliser les valeurs par défaut pour utiliser le traitement de colonne par défaut tel que défini pour cette session.

Sélectionnez le Type approprié pour la colonne :

Détecté

Détermine automatiquement le type de colonne basé sur les données des fichiers.

Booléen

Les valeurs de colonne sont "Vrai" ou "Faux" ou numériques.  Zéro est équivalent à Faux, et n'importe quelle valeur non nulle est Vrai.

Date

La colonne contient des valeurs date-heure dans n'importe quel format usuel.  La comparaison ignore les différences de format.

Numérique

Les valeurs de colonne sont interprétées comme des valeurs numériques réelles.  La comparaison ignore les différences de format.  Par exemple, 0,00 et 0 sont considérées comme identiques.

Texte

La colonne contient des données textuelles.

Cochez Non important pour indiquer que cette colonne est non importante pour votre comparaison.  (Les différences non importantes sont colorées en bleu au lieu de rouge, ou sont toutes ignorées ensemble en utilisant la fonctionnalité Ignorer les différences non importantes .)

Utilisez Tolérance de date pour autoriser dans cette colonne des valeurs de date-heure qui diffèrent par une valeur donnée avant d'être considérées comme une différence importante.

Définir la valeur de Tolérance numérique qui autorise des nombres à différer d'une plus grande quantité avant d'être considérés comme une différence importante.

L'option Est important, sauf pour régit les colonnes qui sont définies ou détectées comme Texte.  Cochez Casse de caractère (ABC = abc) si les différences de casse (ie., capitalisation) sont non importantes.  Cochez Espace blanc pour considérer des différences d'espace blanc (le nombre de blancs avant, après ou entre les mots) comme non importantes.

Options d'opération sur fichier

Outils > Options > Opérations sur fichier

Confirmations

Par défaut, Beyond Compare demande une confirmation avant d'exécuter différentes actions.  Décochez n'importe quelle de ces cases à cocher pour supprimer les dialogues de confirmation.

Confirmations de synchronisation

Sélectionnez Demander, Oui pour tout ou Non pour tout pour contrôler les confirmations qui peuvent apparaître pendant une commande de synchronisation de session de Comparaison de Dossiers.

Comment les filtres affectent les actions de fichier

Par défaut, les filtres de fichier et les filtres d'affichage sont respectés pendant les actions de fichier (telles que Copier, Déplacer et Supprimer), sauf si Supprimer les filtres est actif.  Pour changer ce comportement par défaut, cochez Inclure par défaut les éléments cachés.

Fichiers archives

Des archives sont des fichiers, tels que .zip, qui contiennent d'autres fichiers.  Beyond Compare les affiche comme des dossiers, avec une icône de fermeture éclair, mais les traite par défaut comme des fichiers en les copiant ou en les comparant.  Une fois que vous avez développé l'archive en double-cliquant dessus, les fichiers à l'intérieur seront comparés.  Beaucoup de formats supportent aussi bien la lecture que l'écriture, ainsi vous pouvez utiliser toutes les comparaisons habituelles de fichiers à l'intérieur des archives, incluant la comparaison de contenu, la copie, le renommage, etc.

Formats courants d'archive

 7-zip (*.7z;*.7z.001)

 Beyond Compare Snapshot (*.bcss;*.bcs)

 BZip (*.bz;*.bz2)

 BZipped Tar (*.tbz;*.tbz2;*.tar.bz2)

 Compiled HTML Help (*.chm)

 Debian Package (*.deb)

 Disk Image (*.img;*.iso)

 GZip (*.gz)

 GZipped Tar (*.tgz;*.tar.gz)

 Microsoft Cabinet (*.cab)

 RAR (*.rar)

 Red Hat Package (*.rpm)

 Tar (*.tar)

 Windows Imaging Format (*.wim;*.swm)

 Xz (*.xz)

 Zip (*.zip;*.zipx;*.jar;*.ear;*.war;*.bcpkg)

 

Pour voir la liste complète des formats d'archive, ou changer les extensions de fichier qui leur sont associées, sélectionnez Outils > Options et allez à la page Types d'archive.

Instantanés Beyond Compare

Ces fichiers sont similaires aux autres archives, mais incluent uniquement les noms de fichier, les dates et les tailles sans inclure le contenu du fichier lui-même.  Puisqu'ils sont si petits, vous pouvez enregistrer un Instantané du disque dur complet et le comparer plus tard à lui-même.

Modules d'extension "packer" de Total Commander

Beyond Compare peut aussi utiliser les modules d'extension "Packer" de Total Commander pour supporter des fichiers archives additionnels.  Par exemple, il y a des modules d'extension pour supporter les images disque ISO (.iso) et les packages d'installation Microsoft (.msi).

Pour ajouter un module d'extension d'une partie tierce, aller sur la page Types d'archive, cliquez sur le bouton Ajouter un module d'extension, et suivez les instructions.

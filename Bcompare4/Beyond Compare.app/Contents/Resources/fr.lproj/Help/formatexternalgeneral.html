Paramètres généraux de format externe

Format externe > Général

Pour comparer des fichiers avec une autre application, définissez un format externe.

Le Masque doit être l'ensemble des types de fichier qu'il traite.

La Ligne de commande de la comparaison rapide sera utilisée chaque fois que des résultats de comparaison basée sur des règles sont nécessaires.  Le code de retour doit être 0 pour une correspondance et 1 pour une non-correspondance.

La ligne de commande de la vue de Comparaison sera utilisée lorsqu'une vue de Comparaison sera nécessaire.

La Ligne de commande de la fusion rapide sera utilisée chaque fois que des résultats de fusion basée sur des règles sont nécessaires.  Le code de retour doit être 0 pour une correspondance et 1 pour une non-correspondance.

La Ligne de commande de la vue de fusion sera utilisée chaque fois qu'une vue de fusion sera nécessaire.

La Description peut inclure n'importe quel texte que vous souhaitez.

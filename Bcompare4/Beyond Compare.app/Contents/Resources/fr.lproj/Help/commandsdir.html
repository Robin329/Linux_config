Commandes de Comparaison de Dossiers

Ce qui suit est une description des commandes de Comparaison de Dossiers qui s'ajoutent aux commandes communes.  La plupart de ces commandes sont disponibles comme des boutons de barre d'outils.

Sélectionnez Outils > Options et aller à la page Barre d'outils, etc. pour définir vos propres raccourcis clavier, et cacher ou afficher des commandes dans les menus et barres d'outils.

Le menu Session

Enregistrer la session

Enregistre la session active.

Enregistrer la session sous

Enregistre la session active sous un nouveau nom.

Paramètres de session

Ouvre le dialogue Paramètres de session de la Comparaison de Dossiers.

Verrouillée

Verrouille ou déverrouille la session active.

Effacer la session

Remplace la vue active par une Comparaison de Dossiers vide.

Permuter les côtés

Échange les dossiers de base gauche et droite.

Précédent

Retourne aux dossiers de base comparés (auparavant).

Suivant

Retourne aux dossiers de base comparés (après).

Rechercher un dossier >

  Côté gauche / Côté droit

Recherche un dossier existant du côté spécifié.

Niveau supérieur >

  Côté gauche / Côté droit / Les deux côtés

Change l'un ou l'autre, ou les deux dossiers de base en leur dossier parent.

Rapport de Comparaison de Dossiers

Produit un Rapport de Comparaison de Dossiers.

Informations de Comparaison de Dossiers

Affiche les statistiques de comparaison.

Fusionner les dossiers de base

Ouvre une nouvelle vue de Fusion de Dossiers avec les dossiers de base spécifiés.

Synchroniser les dossiers de base

Ouvre une nouvelle session de Synchronisation de Dossiers.

Comparer les dossiers parents

Ouvre une nouvelle session de Comparaison de Dossiers basée sur les dossiers parents des dossiers de base actuels.

Le menu Actions

Ouvrir un dossier / Fermer le dossier

Ouvre ou ferme le dossier sélectionné dans la comparaison.

Ouvrir les sous-dossiers

Ouvre les dossiers sélectionnés ainsi que leurs sous-dossiers.

Fermer les sous-dossiers

Ferme les dossiers sélectionnés ainsi que tous leurs sous-dossiers.

Définir comme dossier de base

Définit le dossier sélectionné comme le dossier de base pour son côté de la comparaison.

Définir comme base pour l'autre côté

Définit le dossier sélectionné comme le dossier de base pour l'autre côté de la comparaison.

Définir comme dossiers de base

Définit les deux dossiers sélectionnés comme les dossiers de base à gauche et à droite dans la vue de comparaison.  (Pour les ouvrir dans une nouvelle vue, utiliser Ouvrir dans une nouvelle vue à la place.)

Ouvrir

Ouvre les éléments sélectionnés dans une nouvelle vue de comparaison.  Si les éléments sont des fichiers, le type de vue est déterminé par le format de fichier.  Si exactement deux fichiers sont sélectionnés, ils seront comparés entre eux (qu'ils soient ou non en face l'un de l'autre) dans la nouvelle vue.  Si un seul élément est sélectionné, il sera comparé à l'élément en face de lui, s'il existe.

Ouvrir avec

Ouvre les fichiers sélectionnés dans une nouvelle vue en utilisant un type spécifié de session de fichiers, ou ouvre les éléments sélectionnés dans une application externe, telle que définie dans Applications "Ouvrir avec".  Notez que la vue de Comparaison de Dossiers n'est pas automatiquement mise à jour si les éléments sont modifiés par l'application externe.

Ouvrir avec >

  Application associée

Ouvre les fichiers sélectionnés avec l'application associée.

Ouvrir avec >

  Éditeur de texte

Ouvre les fichiers sélectionnés avec l'éditeur de texte intégré.

Ouvrir avec >

  Patch de texte

Ouvre les fichiers sélectionnés avec l'éditeur de vue patch.

Comparaison rapide

Affiche le dialogue Comparaison rapide pour les fichiers sélectionnés.  Si exactement deux fichiers sont sélectionnés, ils seront comparés entre eux (qu'ils soient ou non en face l'un de l'autre).  Si un seul fichier est sélectionné, il sera comparé au fichier en face de lui.

Comparer à

Vous permet de sélectionner un élément, et ensuite de cliquer sur un deuxième élément pour les comparer dans une nouvelle vue.

Aligner

Force les éléments sélectionnés à être côte à côte.  Les dossiers parents des éléments doivent être côte à côte.

Aligner avec

Vous permet de sélectionner un élément, et ensuite de cliquer sur un deuxième élément pour les forcer à être côte à côte.

Comparer les contenus

Compare le contenu des éléments sélectionnés en utilisant une des trois méthodes.

Copier vers la droite / Copier vers la gauche *

Copie les fichiers et les dossiers sélectionnés vers le côté explicitement indiqué.  Normalement associé avec les raccourcis clavier Ctrl+R, Ctrl+L.

Copier vers un côté *

Demande vers quel côté copier les fichiers et les dossiers sélectionnés.

Copier vers l'autre côté

Commande dynamique qui copie les fichiers et les dossiers sélectionnés vers le côté opposé.  Le titre et l'icône apparaît comme Copier vers la droite, Copier vers la gauche ou Copier vers un côté selon le(s) côté(s) sur le(s)quel(s) ils sont sélectionnés.

Déplacer vers la droite / Déplacer vers la gauche *

Déplace les fichiers et les dossiers sélectionnés vers le côté explicitement indiqué.

Déplacer vers un côté *

Demande vers quel côté déplacer les fichiers et les dossiers sélectionnés.

Déplacer vers l'autre côté

Commande dynamique qui déplace les fichiers et les dossiers sélectionnés vers le côté opposé.  Le titre et l'icône apparaît comme Déplacer à droite, Déplacer à gauche ou Déplacer vers un côté selon le(s) côté(s) sur le(s)quel(s) ils sont sélectionnés.

Échanger *

Déplace les fichiers et les dossiers sélectionnés des deux côtés vers le côté opposé.

Copier vers un dossier

Copie les fichiers et les dossiers sélectionnés vers un autre dossier.

Déplacer vers un dossier

Déplace les fichiers et les dossiers sélectionnés vers un autre dossier.

Supprimer

Supprime les fichiers et dossiers sélectionnés.

Renommer

Renomme les fichiers et les dossiers sélectionnés.

Attributs

Change les attributs fichier des fichiers ou des dossiers sélectionnés.

Horodater

Change les horodatages des fichiers ou des dossiers sélectionnés.

Exclure

Exclut le fichier sélectionné, ou le type de fichier, de la session active.

Nouveau dossier

Crée un nouveau dossier.

Copier le nom du fichier

Copie une liste des chemins des éléments sélectionnés vers le presse-papiers.

Ignoré

Supprime les différences dans les éléments sélectionnés.

Rafraîchir la sélection

Rafraîchit les éléments sélectionnés.

Rapport de Comparaison de Fichiers

Produit un rapport de Comparaison de Fichiers pour les fichiers sélectionnés.

Synchroniser >

  Mettre à jour à droite

Copie les éléments plus récents ou orphelins de la gauche vers la droite.

Synchroniser >

  Mettre à jour à gauche

Copie les éléments plus récents ou orphelins de la droite vers la gauche.

Synchroniser >

  Mettre à jour les deux

Exécute une mise à jour bidirectionnelle en copiant les éléments plus récents et orphelins de la gauche vers la droite, et de la droite vers la gauche.

Synchroniser >

  Miroir vers la droite

Remplace tous les éléments différents de droite, supprime tous les orphelins de droite, et copie tous les orphelins de la gauche vers la droite.

Synchroniser >

  Miroir vers la gauche

Remplace tous les éléments différents de gauche, supprime tous les orphelins de gauche, et copie tous les orphelins de la droite vers la gauche.

Contrôle de source

Affiche un sous-menu des actions de contrôle de source (voir ci-dessous).  Apparaît seulement si un fournisseur de contrôle de source a été configuré et tous les éléments sont sous contrôle de source.

Explorateur

Affiche le menu contextuel du gestionnaire de fichiers pour l'(les) élément(s) sélectionné(s).

Sous-menu Contrôle de Source


Ajouter au contrôle de source

Ajoute le fichier au projet.

Enlever du contrôle de source

Enlève le fichier du projet.

Obtenir la dernière version

Obtenir la dernière version du référentiel de contrôle de source.

Obtenir *

Obtenir une version spécifique du référentiel de contrôle de source.

Archiver

Prend vos modifications et les applique au projet.

Extraire

Extrait le fichier du référentiel de contrôle de source et rend votre copie locale modifiable.

Défaire l'extraction

Retire toutes les modifications que vous avez faites à votre copie locale, et la remet en lecture seule.

Afficher les différences

Demande au système de contrôle de version d'afficher la comparaison du fichier avec celle qui se trouve dans le contrôle de version.  Le système de contrôle de version utilisera Beyond Compare seulement si vous lui avez indiqué de faire ainsi.

Afficher l'historique

Affiche l'historique du fichier dans le système de contrôle de version.

Afficher les propriétés

Affiche les propriétés propres au contrôle de version.

Exécuter l'application de contrôle de source

Exécute l'application du contrôle de version.

Le menu Edition

Développer tout

Ouvre tous les dossiers dans la vue.

Réduire tout

Ferme tous les dossiers de la vue.

Sélectionner tout

Sélectionne tous les éléments visibles.

Sélectionner tous les fichiers

Sélectionne tous les fichiers visibles.

Sélectionner les plus récents >

  Côté gauche / Côté droit / Les deux côtés

Sélectionne tous les fichiers visibles qui sont plus récents dans le côté spécifié.

Sélectionner les orphelins >

  Côté gauche / Côté droit / Les deux côtés

Sélectionne tous les fichiers orphelins visibles dans le côté spécifié.

Inverser la sélection

Sélectionne tous les éléments actuellement non sélectionnés, et désélectionne tous les éléments actuellement sélectionnés.

Rafraîchir

Rafraîchit les dossiers ouverts.

Rafraîchissement complet

Rafraîchit la comparaison entière.

Le menu Recherche

Différence suivante

Se positionne sur la différence suivante.

Différence précédente

Se positionne sur la différence précédente.

Rechercher un nom de fichier

Recherche un nom de fichier qui correspond à la chaîne de caractères de recherche.

Rechercher le nom de fichier suivant

Recherche le nom de fichier suivant qui correspond à la chaîne de caractères de recherche.

Rechercher le nom de fichier précédent

Recherche le nom de fichier précédent qui correspond à la chaîne de caractères de recherche.

Le menu Affichage

Afficher tout

N'utilise aucun de filtre d'affichage de n'importe quel côté.

Afficher les différences

Filtre en enlevant les correspondances, basées sur l'état de fichier, des deux côtés.

Afficher les identiques

Filtre en enlevant les plus anciens, les plus récents ou différents, et les orphelins des deux côtés.

Afficher les non-orphelins

Filtre en enlevant les orphelins des deux côtés.

Afficher les différences mais non orphelines

Filtre en enlevant les correspondances et les orphelins des deux côtés.

Afficher les orphelins

Filtre en enlevant les correspondances, les plus anciens, et les plus récents ou différents des deux côtés.

Afficher les plus récents de gauche

Filtre en enlevant les correspondances, les orphelins, les plus anciens de gauche (plus récents ou différents à droite) des deux côtés.

Afficher les plus récents de droite

Filtre en enlevant les correspondances, les orphelins, les plus récents ou différents de gauche (plus anciens à droite) des deux côtés.

Afficher les plus récents de gauche et les orphelins de gauche

Filtre en enlevant les correspondances et les plus anciens à gauche (plus récents ou différents à droite) des deux côtés, et les orphelins du côté droit.

Afficher les plus récents de droite et les orphelins de droite

Filtre en enlevant les correspondances et les plus récents ou différents à gauche (plus anciens à droite) des deux côtés, et les orphelins du côté gauche.

Afficher les orphelins de gauche

Filtre en enlevant les correspondances, les plus anciens, et les plus récents ou différents des deux côtés, et les orphelins du côté droit.

Afficher les orphelins de droite

Filtre en enlevant les correspondances, les plus anciens, et les plus récents ou différents des deux côtés, et les orphelins du côté gauche.

Afficher aucun *

Cache tous les fichiers.  Utile en conjonction avec Toujours afficher les dossiers pour copier des structures de dossier.

Toujours afficher les dossiers

Affiche tous les dossiers, excepté ceux exclus par les filtres de fichier.

Comparer les fichiers et la structure de dossier

Affiche les dossiers contenant des fichiers ou des dossiers qui correspondent au filtre d'affichage actuel.  Les dossiers orphelins suivront les mêmes règles de filtrage que les fichiers orphelins.

Comparer seulement les fichiers

Affiche uniquement les dossiers contenant des fichiers devant être montrés.

Ignorer la structure de dossier

Compare les noms de fichier sans tenir compte des sous-dossiers dans lesquels ils peuvent être.  (Également appelé Aplatir des dossiers.)

Ignorer les différences non importantes

Traite les différences non importantes comme identiques.

Supprimer les filtres

Supprime temporairement les filtres de fichier et les filtres d'affichage.

Colonnes

Affiche ou cache les colonnes de la vue.

Légende

Affiche une fenêtre qui liste ce que les différentes couleurs de dossier signifient.

Journal

Affiche ou cache le panneau du journal.

Barre d'outils

Affiche ou cache la barre d'outils.

* Caché par défaut.

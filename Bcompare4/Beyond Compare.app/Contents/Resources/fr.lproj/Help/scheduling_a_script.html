Planifier un script

Vous pouvez utiliser le processeur de script de Beyond Compare pour exécuter certaines tâches, telle que la synchronisation de dossiers, sans interaction.  Par exemple, pour automatiser une sauvegarde de "C:\Mon Dossier", entrez les commandes de script suivantes dans le Bloc-notes :

load "C:\Mon Dossier" "C:\Mes Sauvegardes"

expand all

select left.newer.files left.orphan.files

copy left->right

 

Enregistrer ce fichier texte comme "C:\Mes Scripts\Sync.txt".

Pour exécuter un script Beyond Compare à un certain moment :

1.Dans le Planificateur de tâches de Windows, sélectionnez Actions > Créer une tâche de base pour avoir un assistant.

2.Dans Actions, sélectionnez Démarrer un programme.

3.Dans Programme/script, sélectionnez l'exécutable Beyond Compare 4.

4.Dans Ajouter des arguments, ajoutez une référence à votre fichier de script :
@Synchroniser.txt

5.Dans Démarrer dans, sélectionnez le dossier du script :
C:\Mes Scripts

Nous recommandons d'inclure une commande de journalisation dans votre script.  Si vous avez des problèmes avec votre script, vérifiez le journal d'erreur des Tâches planifiées : "SchedLgU.txt" dans votre dossier Windows.

Où sont enregistrés les paramètres sur votre ordinateur

Beyond Compare mémorise vos sessions, formats personnalisés de fichier et autres préférences comme des fichiers .xml dans le dossier des paramètres.  Une façon rapide de trouver cet endroit, est de sélectionner Aide > Support et de cliquer sur le lien Explorer le dossier des paramètres dans le bas du dialogue.

Installation standard

Pour Windows, le dossier des paramètres est à l'intérieur de votre dossier de données applicatives.  L'emplacement dépend de votre version de Windows.  Par exemple, sous Windows Vista, le dossier des paramètres est :

C:\Users\<NomUtilisateur>\AppData\Roaming\Scooter Software\Beyond Compare 4

 

Notez que cette localisation diffère pour les différents utilisateurs sur le même ordinateur.

Sur le Mac, le dossier des paramètres est :

~/Library/Application Support/Beyond Compare

 

Si vous voulez installer plusieurs versions majeures, vous pouvez renommer ce dossier en :

~/Library/Application Support/Beyond Compare 4

 

Pour Linux, le dossier des paramètres est :

~/$XDG_CONFIG_HOME/bcompare

 

Si $XDG_CONFIG_HOME n'est pas défini, le dossier des paramètres est :

~/.config/bcompare

 

Si une ancienne version de Beyond Compare a été installée, le dossier des paramètres est :

~/.beyondcompare

 

Si vous voulez installer plusieurs versions majeures, vous pouvez ajouter la version majeure au nom du dossier des paramètres comme ceci :

~/$XDG_CONFIG_HOME/bcompare4

 

Installation portable

Si vous préférez une installation plus portable avec le programme et les paramètres au même endroit, vous pouvez copier votre fichier de paramètres dans le même dossier que le programme.  Les installations portables vous permettent de partager un ensemble de paramètres pour tous les utilisateurs sur un ordinateur, ou de prendre vos paramètres avec vous.  Si vous n'avez pas de paramètres existants à copier, vous pouvez créer un fichier vide nommé settings.xml.

Paramètres de session

Pour modifier les paramètres de session

1.Sélectionnez Session > Paramètres de session .

2.Modifiez les paramètres souhaités dans chaque onglet.

3.La liste déroulante au bas des dialogues contrôle l'étendue de vos changements.  Sélectionnez Utiliser pour cette vue uniquement pour affecter la session active, ou Mettre à jour les défauts de session pour appliquer vos modifications à toutes les nouvelles sessions de ce type.

Des choix additionnels sont affichés pour les sessions filles, des comparaisons engendrées à partir d'une session de dossiers.  Utiliser pour ces fichiers-ci dans la session parente fait que la session active de dossier se souviendra de ces paramètres et les utilisera lorsqu'un couple particulier de fichiers est comparé.  Utiliser pour tous les fichiers dans la session parente fait que ces paramètres seront utilisés pour toute nouvelle session de ce type engendrée par la comparaison courante de dossier.

4.Cliquez sur OK.

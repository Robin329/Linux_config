Importation des paramètres

Outils > Importer les paramètres

Vous pouvez importer des paramètres que vous avez précédemment exportés de Beyond Compare, même d'une plus ancienne version telle que Beyond Compare 2.

Pour importer les paramètres

1.Sélectionnez Outils > Importer les paramètres.

2.Sélectionnez le fichier contenant les paramètres à importer.  Cela peut être un package de paramètres Beyond Compare (.bcpkg) ou un fichier XML.  Ce peut également être un fichier d'export Beyond Compare 2 (fichier zip ou XML).

3.Si le fichier contient différents types de paramètres, cochez les types de paramètres que vous souhaitez importer :

Options de programme inclut la plupart des préférences que vous avez définies dans le dialogue Options.

Couleurs, Fontes inclut toutes vos préférences définies dans la page correspondante du dialogue Options.

Barre d'outils, Raccourcis, Menus inclut vos préférences définies dans la page correspondante du dialogue Options.

État du programme inclut par exemple les positions des fenêtres et les listes des éléments les plus récemment utilisés (Most Recently Used).

Sessions inclut certaines ou toutes vos sessions nommées.

Formats de fichier inclut certains ou tous vos formats personnalisés de fichier.

Profils incluent certains ou tous vos profils nommés, contenant les paramètres pour FTP, etc.

4.Si vous importez des sessions, des formats de fichier ou des profils, cliquez sur Suivant pour sélectionner les éléments nommés à inclure.

Avertissement  Un élément est coloré en rouge si son nom est identique à un de vos paramètres stockés.  Si vous importez un élément en rouge, il remplacera votre session, format de fichier ou profil existant !

Cochez Supprimer toutes les sessions existantes pour effacer toutes vos sessions stockées avant d'importer celles du fichier.  De la même façon, cochez Supprimer tous les formats de fichier existants ou Supprimer tous les profils existants si vous voulez.

5.Cliquez sur Finir pour importer les paramètres.

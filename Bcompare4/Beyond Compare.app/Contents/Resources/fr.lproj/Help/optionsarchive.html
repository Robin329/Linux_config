Options de type d'archive

Outils > Options > Types d'archive

Spécifiez les types de fichier qui doivent être associés avec chacun des formats d'archives supportés.  Sélectionnez un format et entrez les types de fichier associés séparés par des points-virgules dans la colonne Masque.  Vous pouvez le laisser vide pour ne pas supporter ce type de format.  Vous pouvez ajouter le support de formats supplémentaires d'archive en cliquant sur le bouton Ajouter un module d'extension et en suivant les instructions.  Pour plus d'informations, voir Fichiers Archives.

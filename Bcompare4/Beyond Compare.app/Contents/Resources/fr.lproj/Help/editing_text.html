Édition de texte

Chaque panneau de la vue de Comparaison de Textes est un éditeur de texte plein de fonctionnalités.  Vous pouvez entrer et modifier du texte comme n'importe quel autre Éditeur de texte, bien que quand vous le le faites, la comparaison est mise à jour continuellement pour refléter les différences entres les panneaux.

Mode ligne vs. mode caractère

La commande Modification plein écran commute l'éditeur entre le Mode ligne et le Mode caractère.  En mode ligne, la modification se fait sur une base ligne par ligne, ainsi sélectionner, couper, coller, supprimer, etc. affectent des lignes entières.  En mode ligne vous pouvez modifier des lignes individuelles en utilisant la zone Détails de texte.  Le mode ligne supporte des sélections disjointes.  En mode caractère, les panneaux principaux supportent la modification de texte.  Les commandes de sélection et de modification affectent toutes des caractères individuels ou des rangées de caractères.  Le mode caractère ne peut avoir qu'une sélection unique continue.

Modification désactivée

Beyond Compare désactive la modification d'un fichier pour n'importe quelle raison parmi les suivantes :

•La comparaison a rencontré une erreur.

•Le comparaison a été annulée.

•L'emplacement du fichier est en lecture seule (ex. archives 7zip/CHM/RAR, CD-ROMs, DVD-ROMs).

•Le format de fichier conversion ne supporte pas la sauvegarde.

•La mise à jour d'une session de dossier a été désactivée.

•La mise à jour d'une session de fichier a été désactivée.

•Les commutateurs de la ligne de commande ont été désactivés.

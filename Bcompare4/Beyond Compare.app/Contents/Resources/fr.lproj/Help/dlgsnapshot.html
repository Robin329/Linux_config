Enregistrement d'Instantanés

Outils > Enregistrer un Instantané

Pour enregistrer un Instantané

1.Sélectionnez Outils > Enregistrer un Instantané .

2.Entrez le chemin de l'arborescence de dossier à enregistrer.  Si vous êtes actuellement dans une vue Dossier, vous aurez également l'option de sélectionner son Dossier de base de gauche ou Dossier de base de droite.

3.Cochez les cases à cocher appropriées pour également inclure les CRC calculés du contenu de chaque fichier, les données de version ou le contenu des fichiers archives développés.  Ces options rendront plus longue la création de l'Instantané, mais le résultat sera plus approfondi.

CRCs  Inclut les calculs CRC32 dans l'Instantané.

Versions d'Exe  Inclut la version de fichier typiquement trouvée dans les fichiers application tels que .exe, .dll, .ocx, etc.

Contenu d'archive  Inclut la liste développée de contenu du fichier archive dans l'Instantané.

4.Cochez Dossiers vides pour inclure les dossiers vides ou les dossiers dont le contenu complet a été filtré.

5.Cochez Cibles des liens symboliques pour inclure les fichiers et dossiers indiqués par les liens symboliques.

6.Si vous êtes actuellement dans une vue Dossiers, vous aurez également l'option d'inclure les Éléments exclus par les filtres de fichier.  Si cette case à cocher ne l'est pas, l'Instantané utilisera les filtres de fichier des la session de dossiers.

7.Cliquez sur OK et un dialogue vous demandera le nom du fichier et l'emplacement où enregistrer l'Instantané.

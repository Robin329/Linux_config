Commandes de Comparaison de MP3

Ce qui suit est une description des commandes de Comparaison de MP3 qui s'ajoutent aux commandes communes.  La plupart de ces commandes sont disponibles comme des boutons de barre d'outils.

Sélectionnez Outils > Options et aller à la page Barre d'outils, etc. pour définir vos propres raccourcis clavier, et cacher ou afficher des commandes dans les menus et barres d'outils.

Le menu Session

Enregistrer la session

Enregistre la session active.

Enregistrer la session sous

Enregistre la session active sous un nouveau nom.

Paramètres de session

Ouvre le dialogue Paramètres de session de la Comparaison de MP3.

Verrouillée

Verrouille ou déverrouille la session active.

Effacer la session

Remplace la vue active par une Comparaison de MP3 vide.

Permuter les côtés

Échange les fichiers de gauche et de droite.

Recharger les fichiers

Recharge les fichiers affichés.

Recomparer les fichiers

Recompare les fichiers affichés sans rechargement.

Rapport de Comparaison de MP3

Produit le Rapport de Comparaison de MP3.

Informations de Comparaison de MP3

Affiche les statistiques de comparaison.

Jouer les deux

Joue les deux pistes audio.

Comparer des fichiers en utilisant

Ouvre les fichiers affichés dans une nouvelle session en utilisant une classe différente de vue de fichiers.

Comparer les dossiers parents

Ouvre une nouvelle session de Comparaison de Dossiers basée sur les dossiers parents des fichiers actuellement chargés.

Le menu Fichier

Ouvrir un fichier

Ouvre un fichier existant dans le panneau sélectionné.

Ouvrir un fichier du système de fichier *

Ouvre un fichier local existant dans le panneau sélectionné.

Ouvrir un fichier en utilisant un profil *

Ouvre un fichier existant dans le panneau sélectionné en utilisant un profil.

Jouer

Joue une piste audio.

Ouvrir avec

Ouvre le fichier du panneau sélectionné avec une des applications externes Ouvrir avec.

Ouvrir avec >

  Application associée

Ouvre le fichier du panneau sélectionné avec son application associée.

Explorateur

Affiche le menu contextuel du gestionnaire de fichier pour le fichier du panneau sélectionné.

Le menu Edition

Copier

Copie les lignes sélectionnées dans le presse-papiers.

Sélectionner tout

Sélectionne toutes les lignes visibles dans le panneau actif.

Le menu Recherche

Élément différent suivant

Se positionne sur l'élément différent suivant.

Élément différent précédent

Se positionne sur l'élément différent précédent.

Fichiers différents suivants

Ouvre le couple suivant de fichiers avec des différences, fichiers de la session du dossier parent.  (Sessions filles uniquement.)

Fichiers différents précédents

Ouvre le couple précédent de fichiers avec des différences, fichiers de la session du dossier parent.  (Sessions filles uniquement.)

Copier le fichier vers la droite et ouvrir le différent suivant

Copie le fichier entier du côté gauche vers la droite et ouvre les prochains fichiers avec différences de la session de dossiers.  (Sessions filles uniquement.)

Copier le fichier vers la gauche et ouvrir le différent suivant

Copie le fichier entier du côté droit vers la gauche et ouvre les prochains fichiers avec différences de la session de dossiers.  (Sessions filles uniquement.)

Rechercher

Recherche un texte correspondant.

Rechercher le suivant

Recherche l'occurrence suivante de la chaîne de caractères de recherche.

Rechercher le précédent

Recherche l'occurrence précédente de la chaîne de caractères de recherche.

Le menu Affichage

Afficher tout

Affiche tous les éléments, avec ou sans différence.

Afficher les différences

Affiche uniquement les éléments avec des différences.

Afficher les identiques

Affiche uniquement les éléments sans différence.

Afficher aucun *

Cache tous les éléments.

Ignorer les différences non importantes

Traite les différences non importantes comme identiques.

L'élément est important

Marque l'élément sélectionné comme important ou non important.

Espace blanc visible

Affiche ou cache les espaces, tabulations.

Police d'affichage *

Sélectionne la police à utiliser pour afficher les données de fichier.

Augmente la taille de la police d'affichage *

Augmente la taille de la police de rapport.

Diminuer la taille de la police d'affichage *

Diminuer la taille de la police de rapport.

Réinitialiser la taille de la police d'affichage *

Réinitialise la taille de la police des rapports.

Disposition côte à côte

Arrange les panneaux à gauche et à droite l'un de l'autre.

Disposition dessus dessous

Arrange les panneaux au-dessus en au-dessous l'un de l'autre.

Détails en texte

Affiche ou cache le panneau des détails en texte en bas de la vue.

Informations de fichier

Affiche ou cache les panneaux d'information de fichier au-dessus des panneaux.

Barre d'outils

Affiche ou cache la barre d'outils.

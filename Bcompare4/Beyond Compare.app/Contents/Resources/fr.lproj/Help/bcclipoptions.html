Options de Comparaison du Presse-Papiers

Ces options contrôlent le comportement de la Comparaison du Presse-Papiers.

Exécuter au démarrage de Windows

Exécute automatiquement la Comparaison du Presse-Papiers lorsque Windows démarre.

Autoriser les raccourcis clavier

Autorise le raccourci clavier Ctrl+Alt+C pour comparer les deux éléments les plus récents du presse-papiers.

Masque du format du fichier associé

Définit un masque de fichier simple, tel que *.java, à utiliser pour sélectionner un format de fichier approprié pour les comparaisons de texte.  Si il est laissé vide, le fichier par défaut sera utilisé.

Longueur minimale de coupure

Définit le nombre minimum de caractères que le texte doit contenir pour qu'il soit mémorisé.  Peut être utilisé pour empêcher la sauvegarde de mots de passe ou d'autres textes courts.  Mettre à 0 pour mémoriser toutes les copies.

Limites de stockage (clips)

Définit le nombre maximum d'éléments à conserver.

Limites de stockage (MO)

Définit la taille maximale de la mémoire à utiliser pour tous les éléments.

Note  La Comparaison du Presse-Papiers essaiera de retenir les éléments les deux plus récents du presse-papiers, même si la limite d'utilisation de la mémoire est dépassée.

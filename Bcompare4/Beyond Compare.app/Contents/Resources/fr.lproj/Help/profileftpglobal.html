Paramètres globaux de profil FTP

Profil FTP par défaut > Global

Connexions anonymes

Adresse courriel

Lors de connexion à des sites FTP anonymes, il est habituel d'inclure votre adresse courriel comme mot de passe.  Spécifiez l'adresse courriel (réelle ou fictive) à utiliser pour des connexions anonymes.

Authentification de la clé publique SSH

Ces paramètres sont utilisés pour s'authentifier sur un serveur SSH en utilisant une paire de clés publique/privée.  Les clés stockées lors de l'utilisation de l'agent ssh OpenSSH et des agents SSH reconstitués de PuTTY seront utilisées automatiquement.

Fichier de la clé privée SSH

Spécifiez le chemin de la clé privée SSH.  Les formats de clé OpenSSH, SSH.COM, PuTTY et PGP sont supportés.

Créer les clés

Cliquez pour créer une nouvelle paire de clés SSH.

Authentification SSL

Ces paramètres sont utilisés pour s'authentifier sur un serveur FTP SSL en utilisant un certificat de client SSL.

Certificat du client SSL

Spécifiez le chemin du fichier du certificat SSL du client.

Créer le certificat

Cliquez pour créer un nouveau certificat client auto-signé SSL.

Types ASCII

Pour indiquer quels seront les fichiers à transférer en ASCII, ajoutez et supprimez des extensions de cette liste.

Note  Si vous avez besoin d'aide pour les paramètres de cette page, consultez votre administrateur système.

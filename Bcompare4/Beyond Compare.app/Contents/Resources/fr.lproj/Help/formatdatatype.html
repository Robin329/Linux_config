Paramètres de type de format de tableaux

Format tableau > Type

L'onglet Type définit la syntaxe du fichier de données.

Fichiers texte délimités

Sélectionnez Délimité si les champs sont séparés par un caractère délimiteur tel qu'une tabulation ou une virgule.

Cochez Détecter pour que le programme détermine automatiquement le délimiteur.  Sinon, sélectionnez les Délimiteurs qui séparent les champs et sélectionnez le Qualificateur de texte qui entoure optionnellement les champs (nécessaire pour les données qui contiennent le caractère délimiteur).

Cochez Traiter des délimiteurs consécutifs comme un seul si un délimiteur peut consister en de multiples caractères délimiteur.  Par exemple, si le fichier de données contient des colonnes séparées par des espaces blancs, vous devriez cocher cette option et sélectionner Espace comme étant le caractère délimiteur.

Décochez Traiter les espaces blancs voisins comme partie du délimiteur si tout espace blanc à côté d'un délimiteur doit être considéré comme faisant partie du champ de données.

Fichiers texte à position fixe

Sélectionnez Fixé pour les fichiers où les champs sont définis par position sur la ligne.

Entrez les Largeurs de colonne dans une liste séparées par des points-virgules.

Tous les types

Sélectionnez l'option La première ligne contient pour contrôler si la première ligne du fichier doit être traitée comme un cellule de donnée ou comme les noms des colonnes.  Normalement, la Comparaison de Tableaux essaie de le détecter, mais vous pouvez passer outre ici.

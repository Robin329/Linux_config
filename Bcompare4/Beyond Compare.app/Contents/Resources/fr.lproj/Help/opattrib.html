Attributs

Actions > Attributs

Dans le menu Actions, la commande Attributs modifie divers attributs du système d'exploitation des fichiers et dossiers sélectionnés.

Pour les systèmes de fichier Windows, les attributs DOS (Lecture seule, Caché ou Archive) sont modifiés.  Vous ne pouvez pas modifier l'attribut Système.

Pour les systèmes de fichiers Mac et Linux, les autorisations d'accès propriétaire et groupe sont modifiées.

Si un dossier est sélectionné, ses attributs sont modifiés, mais pas ceux de son contenu.  Pour modifier les attributs du contenu d'un dossier, vous devez développer le dossier et sélectionner son contenu.

Les cases à cocher montrent initialement l'état actuel des éléments sélectionnés.  Si une case à cocher n'est pas cochée, cet attribut est interdit pour tous les fichiers sélectionnés.  Si une case à cocher est cochée, l'attribut est disponible pour tous les fichiers sélectionnés.  Si une case à cocher est cochée et grisée, l'attribut est uniquement disponible pour quelques-uns des fichiers sélectionnés.

Décochez une case à cocher pour effacer l'attribut des fichiers sélectionnés, ou cochez-la pour positionner l'attribut.

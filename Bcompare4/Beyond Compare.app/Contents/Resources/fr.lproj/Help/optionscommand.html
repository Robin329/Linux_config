Options de Barre d'outils, Raccourcis, Menus

Outils > Options > Barre d'outils, etc.

Ces options vous permettent de contrôler quelles commandes apparaissent dans les menus et dans les barres d'outils, et vous permettent de définir des raccourcis clavier pour des commandes.  Comme les menus et les barres d'outils changent selon le type de fenêtre, vous avez besoin de sélectionner le type de session appropriée.

Dans le champ de recherche , entrez le nom ou la description de la commande que vous rechercher pour limiter la liste des commandes.

Sélectionnez dans la liste une commande à éditer.  La description de la commande apparaît en dessous de la liste.

Les cases à cocher dans les colonnes Menu et Barre d'outils contrôlent où apparaissent les commandes.

Pour modifier le raccourci clavier, mettez le focus sur le champ Raccourci et pressez la touche souhaitée.  Cliquez sur le bouton Enlever pour enlever le raccourci attribué ou sur le bouton Défaut pour revenir au paramétrage par défaut usine.  Si un raccourci clavier est déjà attribué autre part, un avertissement apparaît et vous avez la possibilité de le réattribuer.

La plupart des types de session ont des commandes de filtre d'affichage disponibles.  Le Mode de filtre d'affichage peut être changé en Liste déroulante, Favoris ou Bascules.  Cela peut également être modifié via un clic-droit sur la barre d'outils de la vue.

Grands boutons avec libellés peut également être modifié via un clic-droit sur la barre d'outils de la vue.

Pour une documentation des commandes spécifiques, voir Référence des commandes.

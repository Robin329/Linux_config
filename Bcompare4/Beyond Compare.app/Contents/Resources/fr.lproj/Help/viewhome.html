Vue d'ensemble de la page d'accueil

Lorsque vous ouvrez une nouvelle fenêtre ou onglet, Beyond Compare affiche la vue Accueil :

Depuis la vue Accueil, vous pouvez :

•Démarrer une nouvelle session de comparaison de n'importe quel type

•Ouvrir une de vos sessions enregistrées auparavant

•Ouvrir une session sauvegardée automatiquement

•Rechercher une session

•Gérer les paramètres de vos sessions enregistrées

•Modifier les paramètres par défaut à utiliser pour de nouvelles sessions

Vous pouvez aussi glisser-déposer des fichiers et des dossiers d'un gestionnaire de fichiers dans cette vue pour les comparer.  Glisser-déposer sur un des grands boutons pour démarrer une session de ce type spécifique.

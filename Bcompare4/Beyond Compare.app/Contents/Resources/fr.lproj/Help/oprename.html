Renommer

Actions > Renommer

Dans le menu Actions, la commande Renommer renomme les fichiers et les dossiers sélectionnés.

Dans la sous-section Action, vous pouvez sélectionner un renommage soit Compatible DOS ou Expressions régulières.

Renommage DOS

Pour renommer des fichiers en utilisant des caractères génériques DOS, sélectionnez Compatible DOS dans le dialogue Renommer.  Dans le champ Nouveau masque, entrez le nouveau nom.

Si plusieurs fichiers ou dossiers sont sélectionnés, un masque de base est calculé, masque qui représente l'intersection commune de tous les noms.  Un astérisque (*) est utilisée pour représenter un ou plusieurs caractères, tandis qu'un point d'interrogation (?) est utilisé pour représenter un seul caractère.  Laissez ces caractères seuls, mais changez les autres pour produire des noms uniques pour chacun des fichiers ou des dossiers sélectionnés.

Exemples :

Supposez que vous avez sélectionné les fichiers suivants à renommer :

    abc1.txt

    abc2.txt

    abc3.txt

 

Si vous entrez "abc?.bak" dans le nouveau masque, les fichiers seront renommés en :

    abc1.bak

    abc2.bak

    abc3.bak

 

Si vous entrez "xyz?.txt" pour le nouveau masque, les fichiers seront renommés en :

    xyz1.txt

    xyz2.txt

    xyz3.txt

 

Renommage via une expression régulière

Pour renommer des fichiers en utilisant des expressions régulières compatibles Perl, cochez Expressions régulières dans le dialogue Renommer.  Dans le champ Ancien masque, entrez une expression régulière qui correspond au nom de fichier original.  Dans le champ Nouveau masque, entrez un modèle utilisé pour générer un nouveau nom de fichier.

Exemples :

Supposez que vous avez sélectionné les fichiers suivants à renommer :

    abc1.txt

    abc2.txt

    abc3.txt

 

Si vous entrez "txt" pour l'ancien masque et "bak" pour le nouveau masque, les fichiers seront renommés en :

    abc1.bak

    abc2.bak

    abc3.bak

 

Si vous entrez ".*(\d\.txt)" pour l'ancien masque et "xyz$1" pour le nouveau masque, les fichiers seront renommés en :

    xyz1.txt

    xyz2.txt

    xyz3.txt

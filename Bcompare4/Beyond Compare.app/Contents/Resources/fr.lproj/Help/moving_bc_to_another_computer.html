Migrer Beyond Compare vers un autre ordinateur

Pour copier ou déplacer Beyond Compare vers un ordinateur différent

1.Sur le nouvel ordinateur, téléchargez et installez la dernière version de Beyond Compare.

2.Entrez votre clé d'enregistrement dans la nouvelle installation.

3.Exécutez Beyond Compare sur l'ancien ordinateur et sélectionnez Outils > Exporter les paramètres pour enregistrer tous les paramètres souhaités dans un fichier.

4.Copiez ce fichier de paramètres vers le nouvel ordinateur.

5.Exécutez Beyond Compare sur le nouvel ordinateur et sélectionnez Outils > Importer les paramètres pour charger le fichier des paramètres.

Note  Si vous possédez Beyond Compare 2.0 ou plus ancienne sur le vieil ordinateur, vous n'avez pas de commande d'exportation des paramètres.  Vous pouvez contacter Scooter Software pour de l'assistance dans le transfert de vos paramètres.

Paramètres d'exportation

Outils > Exporter les paramètres

Vous pouvez enregistrer sélectivement vos paramètres du programme pour les transférer vers un autre ordinateur, mettre à niveau vers une nouvelle version, ou les partager avec une autre utilisateur de Beyond Compare.

Pour exporter vos paramètres

1.Sélectionnez Outils > Exporter les paramètres.

2.Sélectionnez les types de paramètres que vous voulez exporter :

Options de programme inclut la plupart des préférences que vous avez définies dans le dialogue Options.

Couleurs, Fontes inclut toutes vos préférences définies dans la page correspondante du dialogue Options.

Barre d'outils, Raccourcis, Menus inclut vos préférences définies dans la page correspondante du dialogue Options.

État du programme inclut par exemple les positions des fenêtres et les listes des éléments les plus récemment utilisés (Most Recently Used).

Sessions inclut certaines ou toutes vos sessions nommées.

Formats de fichier inclut certains ou tous vos formats personnalisés de fichier.

Profils incluent certains ou tous vos profils nommés, contenant les paramètres pour FTP, etc.

Informations système inclut les informations concernant votre ordinateur et votre installation de Beyond Compare.  C'est utile dans un but de support pour nous aider à résoudre des problèmes.

3.Si vous exportez des sessions, des formats de fichier ou des profils, cliquez sur Suivant pour sélectionner les éléments nommés à inclure.

4.Cliquez sur Finir pour exporter les paramètres.  Un package de paramètres (.bcpkg) sera créé.

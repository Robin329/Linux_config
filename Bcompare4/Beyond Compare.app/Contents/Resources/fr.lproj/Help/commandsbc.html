Commandes communes

Le menu principal change en fonction du type de session qui est active.  Toutefois, les commandes listées ci-dessous sont communes à travers les différents types de vue.

Sélectionnez Outils > Options et aller à la page Barre d'outils, etc. pour définir vos propres raccourcis clavier, et cacher ou afficher des commandes dans les menus et barres d'outils.

Le menu Session

Nouvelle session

Ouvre une nouvelle vue avec une session vide du type spécifié.

Nouvel onglet

Ouvre un nouvel onglet dans la fenêtre active et affiche la vue Accueil.

Nouvelle fenêtre

Ouvre une nouvelle fenêtre et affiche la vue Accueil.

Ouvrir une session

Ouvre une nouvelle vue et affiche la vue Accueil.

Charger un espace de travail

Liste les espaces de travail enregistrés.  En sélectionner un, ferme toutes les vues ouvertes et restaure la configuration sélectionnée.

Charger un espace de travail  >

  Gérer les espaces de travail

Vous permet de renommer ou de supprimer des espaces de travail enregistrés.

Enregistrer l'espace de travail sous

Enregistre la configuration active des fenêtres et des onglets comme un espace de travail nommé.

Fermer l'onglet

Ferme l'onglet actif.

Quitter

Quitte Beyond Compare, en demandant d'enregistrer toutes modifications.  Toutes les fenêtres d'instance se fermeront.

Le menu Outils

Options

Ouvre le dialogue Options pour définir les préférences du programme.

Formats de fichier

Ouvre le dialogue Formats de fichier pour gérer les paramètres de format de fichier.

Profils

Ouvre le dialogue Profils pour gérer les paramètres de profil.

Intégration de contrôle de source

Ouvre le dialogue Intégration du Contrôle de Source.  Disponible uniquement dans la version Windows.

Exporter les paramètres

Ouvre le dialogue Exportation des paramètres pour enregistrer sélectivement les paramètres et les sessions dans un fichier.

Importer les paramètres

Ouvre le dialogue Importation des paramètres pour charger sélectivement les paramètres et les sessions d'un fichier.

Restaurer les défauts usine

Ouvre le dialogue Restaurer les défauts usine pour remettre sélectivement des paramètres à leur valeur par défaut.

Enregistrer un Instantané

Ouvre le dialogue Enregistrer un Instantané pour créer un cliché de la structure de dossier.

Modifier un fichier texte

Ouvre une nouvelle vue d'Édition de texte.

Visualiser le patch

Ouvre une nouvelle vue de Patch de texte.

Le menu Aide

Sommaire

Affiche le sommaire de l'aide.

Aide contextuelle

Affiche la page d'aide relative à ce sur quoi vous êtes focalisé.

Scooter Software sur le Web

Ouvre le site web de Scooter Software dans votre navigateur.

Vérifier les mises à jour

Vérifie si une nouvelle version de Beyond Compare est disponible.

Support

Ouvre le dialogue Support qui liste les instructions pour rapporter des problèmes.

Entrer la clé

Ouvre le dialogue Enregistrement pour accepter votre clé de licence.

À propos de Beyond Compare

Affiche la boîte À propos avec la version du programme et l'information de licence.

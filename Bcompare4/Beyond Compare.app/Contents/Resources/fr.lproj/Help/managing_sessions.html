Gestion des sessions

Beyond Compare enregistre automatiquement les sessions les plus récentes de telle façon que vous pouvez y revenir facilement.  Si vous voulez retenir une session indéfiniment, vous pouvez l'enregistrer avec un nom descriptif.  Vous pouvez également organiser vos sessions nommées dans des dossiers de session.

Utilisez la vue Accueil pour gérer toutes vos sessions à un seul endroit.  Dans l'arborescence Sessions, vous pouvez faire un clic-droit et sélectionner Modifier pour afficher le panneau de gestion.  Cliquez sur le bouton Épingler la gestion de session ou cocher Affichage > Afficher la gestion de session pour le forcer à toujours être affiché dans la vue d'Accueil.  Supprimer et Renommer sont aussi disponibles dans le menu clic-droit pour les sessions sauvegardées.

Une session nommée peut être verrouillée pour se protéger contre des modifications accidentelles.  Clic-droit sur lui dans l'arborescence Sessions, et sélection de Verrouiller.

Une fois la session ouverte dans une vue, le menu Session contient les commandes usuelles pour la manipuler.  Si vous modifiez les paramètres primaires de session nommée, on vous demandera de l'enregistrer à la sortie.

Pour changer les paramètres par défaut de session

Dans l'arborescence des Sessions de la vue Accueil, sous la branche Nouvelle, sélectionnez un type de session.  Clic-droit sur lui, et sélection d'Edit pour afficher le panneau de gestion.  Les modifications de ces paramètres par défaut seront maintenant utilisées pour créer de nouvelles sessions de ce type.

Nouveau dossier

Actions > Nouveau dossier

Dans le menu Actions, la commande Nouveau dossier vous permet de créer un dossier à la position actuelle.  Si vous avez un fichier ou un dossier fermé sélectionné, le nouveau dossier sera créé avec le même parent que l'élément sélectionné.  Si vous avec un dossier ouvert de sélectionné, le nouveau dossier sera créé comme sous-dossier de celui qui est sélectionné.

Si les deux côtés sont sélectionnés, un nouveau dossier sera créé des deux côtés.

Vous pouvez créer un ficher archive en incluant l'extension appropriée (telle que .zip) dans son nom.

Pour créer un nouveau dossier à utiliser comme dossier de base, entrez le chemin d'accès pour le nouveau dossier dans les contrôles texte à gauche ou à droite et pressez la touche Entrée.  Dans le dialogue d'avertissement qui apparaît, cliquez sur le bouton Créer.

Options de couleur et de fontes sur la vue Dossiers

Outils > Options > Couleurs, Fontes > Vues Dossiers

Ces options vous permettent de personnaliser l'ensemble des couleurs et des polices utilisées dans les vues Dossiers.

Police

Par défaut, la police utilisée pour l'affichage de dossier est la même que celle utilisée par le système d'exploitation.  Pour la changer, décochez Utiliser la fonte système et cliquez sur le bouton Police.

Sélection

L'élément Sélection contrôle le traitement du texte sélectionné.  Par défaut, Beyond Compare utilise une couleur bleu moyen pour les sélections, de telle façon que la coloration des différences ne soit pas obscurcie.  Si vous préférez le traitement standard du système d'exploitation pour les sélections, cochez Utiliser le système.  Les fichiers sélectionnés perdront leur coloration de différence, quoique l'indicateur de couleur à gauche de chaque nom de fichier sera coloré de manière adéquate.

Divers

Sélectionnez les couleurs à utiliser pour le Filtrage d'éléments (lorsque Supprimer les filtres est utilisé), et pour les éléments dont l'état de comparaison est Inconnu.  Cochez Utiliser les bandes colorées pour teinter l'arrière-plan de chaque autre ligne dans la comparaison.

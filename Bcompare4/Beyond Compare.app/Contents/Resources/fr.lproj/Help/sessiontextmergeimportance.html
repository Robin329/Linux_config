Paramètres d'importance de la Fusion de Textes

Session > Paramètres de session > Importance

Ces paramètres contrôlent quelles sont les différences que vous considérez comme importantes dans la session active.  Les différences importantes seront colorées en rouge dans la comparaison.  Les différences non importantes seront colorées en bleu (ou non surlignées du tous si Affichage> Ignorer les différences non importantes est coché).

Non-importance

Le texte qui correspond aux éléments définis dans la liste sont considérés comme non importants.

Éléments de grammaire

Cochez les cases dans la liste d'éléments qui sont importants pour la comparaison.

Espace blanc au début contrôle si un espace blanc en début de ligne est important.

Espace blanc intérieur contrôle si un espace blanc en milieu de ligne est important.

Espace blanc à la fin contrôle si un espace blanc en fin de ligne est important.

Tout le reste se réfère au texte non blanc qui n'est pas encore défini comme un élément de grammaire.

Cochez Casse de caractère si le texte "tout le reste" est sensible à la casse.  (La sensibilité à la casse des éléments de grammaire est contrôlée par la grammaire des formats de fichier.)

Cliquez sur le bouton Modifier la grammaire pour examiner ou modifier les définitions de grammaire dans le format de fichier sélectionné.

Divers

Pour considérer une ligne blanche supplémentaire (ou une ligne insérée avec uniquement du texte non important) comme une différence importante, cochez Les lignes orphelines sont toujours importantes.

Beyond Compare ignore normalement les différences des styles de fin de ligne tels que CR+LF dans des fichiers texte Windows vs. un LF seul dans les fichiers Unix.  Pour trouver des différences dans les fins de lignes sur une base de ligne par ligne, cochez Comparer les fins de ligne (PC/Mac/Unix).  (Pour examiner les fins de ligne, cochez Affichage > Espace blanc visible .)
